Function synopsis

COPYSUBS - copies a list of subplots into a new figure

Function comments

 
% 
% copysubs(sf, tf, submatsize, linidx) copies subplots 
% from the source figure with handle <sf> to the same 
% positions in target  figure with handle <tf>. <submatsize> 
% gives the number of rows and columns in the subplot arrangement 
% of the source figure. <linidx> gives the subplot index 
% of the axis objects to copy. Note that in this scheme 
% subplots are counted in the  usual matlab notation along 
% columns. If the target figure does not exist a new one is 
% created, as are non existing subplots. 
% 
% example 
% sf = figure; 
% subplot(2,2,1); plot(rand(1,10)); 
% subplot(2,2,2); plot(rand(1,50)); 
% copysubs(sf, figure, [2,2], [1 4 ]) 
% 
% This will copy the newly created subplots to a different figure and 
% create an empty axis in position [2,2,3] 
% U. Egert 8/98
