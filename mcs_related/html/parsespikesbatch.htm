parsespikesbatch - GUI to parse several MCRacks spike data files

Function comments

 
% parsespikesbatch opens a GUI to adjust the settings necessary to 
% parse all MCRack spike cutout files in a directory (and thats only 
% those) and save the results in mat-files. 
% 
% As an option principle component vectors that were either calculated 
% and stored previously or are calculated from one MCRack file in the 
% set may be reused. 
% 
% Note that you need to reload the results files to have the correct 
% data set in your workspace.
