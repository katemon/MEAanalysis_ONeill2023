% [lh, th,c, linevalues]=makesurf2contour(varargin) 
% creates a contourplot from the color or height data of a 
% surface plot. <lh> are handles to the contour lines (as returned 
% by countour.m, <c> is the corresponding countour matrix. <th> 
% are handles to the contour line labels or the list of contour 
% levels. <linevalues> are the levels at which lines are drawn, only 
% one value per level is returned, so this vector may be shorter 
% than <lh> if no 'linevalues' are given. 
% 
% Various parameters can be adjusted by parameter/value pairs: 
% 
% parameter         def. value 
% 'sourcefigh'      gcf     handle to the source data obj., can be 
%                           a handle to either a figure, axis or surface obj. 
% 'Nlines'          10      no. of lines in the contour plot 
% 'linevalues'      0       set the lines to the values in this vector 
% 'targetfigname'   'surf2contour' 
% 'datasource'      'color' surface 'height' or 'color' as source values 
% 'linecolor'       0       any color value or 0 for default 
% 'skipNlines'      0       no. of lines to skip when labeling contours 
% 'fontsize'        10      font size for text-labels 
% 'fontname'        arial   font name for these 
% 'labellines'      0       0 to print a list of levels, 1 to label contour lines. 
% 'roundLineVals'   0       any integer to round the levels to the nearest 
%                           decimal, e.g. 1 for nearest integer, 2 to 
%                           nearest tenth, -2 to nearest decade 
% 
% U. Egert 10/01
