Function synopsis

plotmeagrid_objects - create a MEA raster for electrode selection

Function comments

 
% [figh, axh, doth] = plotmeagrid_objects(meaDim, skipcid); <meaDim> is a 
% vector defining the number of rows and columns of the grid. <skipcid> 
% (may be empty) 
% indicates linear indices to positions that should be marked by a '.' 
% only. Clicking on circles will turn them red or blank. 
% 
% <figh>, <axh>, and <doth> are handles to figure, axis and circles 
% respectively (not those with '.' markers). 
% 
% The 'finish' button will send a list of marked electrodes with electrode 
% names defines in colum/row format to a variable 'Ename' in the calling 
% workspace. A rubberband utility allows to select by dragging a rubberband 
% box across several electrodes. 
% 'Ename'  will be a string when only one pos. is marked, it will be a 
% cell array of strings when more than one position is marked. If no 
% positions are marked 'Ename' will be an empty string. 
% 
% Known bugs: To start the rubberband utility you need to click on the 
% space between the circles. The rubberband has to start ON a circle. 
% Starting above and left to the circles  center will include circles on 
% the top an left boundary of the box. Starting on or below the center will 
% exclude these, execept for the start circle. 
% 
% Note that the box adds circles, it does not invert ones already marked. 
% 
% 
%
