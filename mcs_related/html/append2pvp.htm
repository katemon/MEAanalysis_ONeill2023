Function synopsis

append2PvP - append parameter/value pairs to an existing list

Function comments

 
% pvp = append2PvP(pvp, varargin); 
% Append a parameter value pair to an existing list.  <pvp> may have 2 rows where 
% the first row holds the parameter names and the second the corresponding 
% values. Alternatively <pvp> can be a cell array with a single row or column 
% whose elements alternatingly hold parameters and values, starting with a 
% parameter in <pvp>{1}. <varargin> is a list of comma-separated variables (not strings). 
% The names of these variables in the calling workspace will become the 
% parameter strings, their content will be the values. The input variable 
% <pvp> may be an empty cell. 
% 
% Note that the parameter must not be the result of a calculation or and 
% expression like a(1)! 
% 
% (c) U. Egert 2002
