% sortspx4(en, subpop) 
% 
% Plots the spike shape parameters stored in PARMAT against 
% each other. The resulting multi-subplot figure is the basis 
% for routines used subsequently to view spike shapes and groups, 
% define clusters and assign cluster IDs and new electrode names 
% to the spikes selected. New names will be created by using 
% the old name and appending the next available letter in alphabetical 
% sequence, e.g. #76 will become # 76b if # 76a already exists. 
% Clusternumbers will be assigned accordingly and stored in 
% the global variable CLUSTERNUMBER. These functions are initiated 
% through GUI. 
% 
% This is not a standalone function and requires datasets based 
% on MCRack records and the MEATools toolbox. 
% 
% U. Egert 
% See also MEATOOLS CHKDIST2 SORTGUI1 SORTCTRL
