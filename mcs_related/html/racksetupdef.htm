racksetupdef - dialog to edit the RACKSETUP structure

Function comments

 
% racksetupdef(D) serves to define or modify some settings of the 
% RACKSETUP structure describing the recording conditions of a 
% file. Called without input arguments racksetupdef will open a 
% dialog box in which the necessary parameters may be set manually. 
% This is useful when the data are store in a matrix that was not 
% recorded with MCRack. If the RACKSETUP structure already exists 
% and is global its entries are used as defaults. racksetupdef in 
% this case is a tool to modify the settings. Alternatively, 
% racksetupdef called with a datastream object <D> will fill RACKSETUP 
% with the information available in D. 
% 
% (c) U. Egert 4/99
