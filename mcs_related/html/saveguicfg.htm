SAVEGUICFG - saves GUI to a structure and file

Function comments

 
% 
% saveguicfg(cfgnametag,fh) copies the userdata fields of GUI 
% <fh> to a structure and saves this 
% structure to a file. The structure name is taken from the 
% string of an object in <fh> with the 'tag' set to <cfgnametag>. 
% If <fh> is omitted the current callback figure is used. 
% Use 'loadguicfg' to load such a file into a GUI. 
% 
%
