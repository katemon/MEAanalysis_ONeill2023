% The 'movie tool' shows a MEA geometry oriented pseudocolor movie of either 
% the data in D, or of an 64xN array. Values are coded in color and can be 
% viewed in 2D or 3D display with different degrees of interpolation and 
% from various perspectives. The tool can not only be used to view raw data 
% is useful to visualize pre-processed data as well, e.g. response-amplitudes 
% across trials or PSTH data from MEA records. 
% 
% 
% The resulting matlab variables then need be reformated into 64xN arrays 
% for visualization or statistical analysis. 
% 
% To avoid crashes use the LFP-Tools in the following sequence: 
% - Create the datastream object D using the file loader, 
% - explore different sweeps in a trace like fashion using the 
%   plot features of the LFP extraction tool, 
% - view the data in a tissue oriented fashion with the movie tool, 
% - extract specific aspects from sets of sweeps  again with the LFP 
%   extraction tool, 
% 
% LFP-Tools were created by U. Egert, A. Brandt, and Th. Knott 
% 
%

%
