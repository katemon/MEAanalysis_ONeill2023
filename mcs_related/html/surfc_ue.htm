Function synopsis

SURFC_UE - combination surf/contour plot, mod. SURFC

Function comments

 
%   SURFC_UE(...) is the same as SURF(...) except that a contour plot 
%   is drawn beneath the surface. SURFC_UE differs form SURFC in 
%   that it (1) returns the handles to the surface and the contours 
%   in separate vectors <sh> and <ch>. It (2) further allows modification 
%   of defaults by parameter value pairs. Specifically 
%   'nlines' may be used to indicate the number of contours drawn. 
% 
%   modified by U. Egert 2/99 
% 
%   See also SURF, SHADING, CONTOURS, SURFC.
