Function synopsis

normaxes - normalizes x- and y-data of line objects to mean==1

Function comments

 
% normaxes by itself normalizes the x- and y-data of all lineobjects 
% in all subplots of the current figure such that (a) their means are 
% 1 and (b) their axes scalings are equal. 
% 
% normaxes(fh, ah) with the optional will do the same for line 
% objects in subplot(ah) in figure(fh) with <ah>, <fh> being graphic 
% handles. 
% If <fh> is empty the current figure will be used. 
% If <ah> is empty the current axes will be used. 
% If <ah> is a string all axes will be used. 
% 
% (c) U. Egert 9/2000
