Function synopsis

MCS2GDF - convert SOURCECHANNELNUMBER/SPIKETIME to gdf

Function comments

 
% gdfmat = mcs2gdf(idx) converts column vectors giving the ChannelID 
% and the spiketime to an N x 2 matrix giving ChannelID in the 1st and 
% event times in the 2nd column. Times are maintained in full temporal 
% resolution which differs from the original gdf format with .5 ms temporal 
% resolution. 
% 
% <idx> points to rows in SOURCECHANNELNUMBER and SPIKETIME, 
% i.e. to events. Both variables must to be global in the base 
% workspace. If <idx> is set to 'all', all spikes are used. 
% i.e. to spikes. 
% 
% If a variable CLUSTERNUMBER exists in the global workspace the gdf 
% 'neuron id' is calculated as SOURCECHANNELNUMBER*100+CLUSTERNUMBER. 
% otherwise 'neuron id' is SOURCECHANNELNUMBER. 
% 
% If the parameter/value pair 'fname', <outfile> is given the resulting 
% matrix will be stored to that file in ASCII format, columns separted 
% by tabs. 
% 
%  'fname'        target file name 
%  'outtimeunits' time units used in the target file given in fractions of seconds 
%                 e.g. 1e-6 for microseconds, 1e-3 for milliseconds (def. 1e-6) 
%  'intimeunits'  time units used in the source variable, as above (def. 1) 
% 
% (c) U. Egert 5/98 
% 
% See also WRITE2GDF
