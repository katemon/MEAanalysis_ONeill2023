LFP_EXTRACT_PLOT - plot results from MCRack sweep evaluation on a trace

Function comments

 
% lfp_extract_plot(rawdata, filtdata, fxidx, paramdata, fitrange, Trigidx, varargin) 
% 
% <paramdata>  must be as described in LFP_EXTRACT.m 
% 
% <rawdata>    unfiltered data set (single trace) 
% <filtdata>   filtered data set 
% <fxidx>      index offset of the fitted range 
% <paramdata>  parameter data to show 
% <fitrange>   the range evaluated 
% <Trigidx>    indices for the trigger used 
% 
% the following parameters can be modified with parameter/value pairs 
% 
% 'subplotcolumns' no. of columns on a MEA (default 8) 
% 'subplotrows'    no. of rows on a MEA    (default 8) 
% 'samplingperiod' sampling period in <samptimeunits> (default 40) 
% 'samptimeunits'  time units for the above in fractions of seconds, e.g. 1e-6 for microseconds 
% 'showpolynom'    1 to show a polynomial fit to the fitrange (default 0) 
% 'trignumber'     no. of trigger indices (default length(Trigidx)) 
% 'dispTimeUnits'  time units to use for displays (default 1e-3) 
% 'Ymin'           minimum to use for the y-axis 
% 'Ymax'           maximum to use for the y-axis 
% 
% Defaults are set extensively to allow for future modification using MxN MEAs 
% the MEA geometry must be described in MEAINFO 
% 
% U. Egert
