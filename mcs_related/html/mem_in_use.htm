mem_in_use - returns the memory occupied by the calling workspace

Function comments

 
% [L, G, S] = mem_in_use(space) returns the memory occupied by the workspace 
% <space>, which is the calling workspace by default. 
% Occupation is the sum of the size of all variables. <L> is the space occupied by 
% local variable, <G> that of global variables, and <S> the sum of unique 
% variables in both.
