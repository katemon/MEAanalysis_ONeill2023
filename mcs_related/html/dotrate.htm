Function synopsis

DOTRATE - plot cell-values and linear regressions (MEATools component)

Function comments

 
% DOTRATE(r,rl) plots the values of the cells in <r(2:end)> against r(1). 
% <rl> is a vector with each element giving the length of the corresponding cell 
% in <r>. It also plots a linear regression line into the data. 
% <rc> is the correlation coefficient of this linear regression. 
% <r(1)> is reused to calculate regressions for the other elements. 
% 
% You may use the following parameter/value pairs to modify the figure settings: 
% 
% 'colorseq'      the color sequence used for sucessive data sets, def: 'brgckym' 
% 'binwidth'      the time across which the rates were calculated, i.e. the leading 
%                 and trailing date ranges that should be excluded from the analysis, 
%                 given in seconds, def. 10 sec 
% 'figurename'    the name for the figure, def: 'rate corr. dot-plot' 
% 
% (C) U. Egert 4/98
