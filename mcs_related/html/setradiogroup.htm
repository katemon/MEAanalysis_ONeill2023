Function synopsis

SETRADIOGROUP - sets exactly one of a group of radiobuttons

Function comments

 
% Syntax 
% activestr = setradiogroup(tagstr, activestr, fh), all input 
%    arguments are optional. 
%    If none are given the values of all radio buttons with the 
%    same tag as the GCBO except GCBO are set to 0. All user 
%    data will have the fields <tag> = <tagstring> and <active> 
%    giving the string of the active button in the group. 
% 
%    If <tagstr> and <activestr> are given the 'Value' property 
%    of radiobuttons of the current figure with the tag <tagstr> 
%    will be set to 0 except the one with its 'string' property 
%    set to <activestr>. 'userdata' will be set as above. Giving 
%    figure handle <fh> will use that figure. 
% 
% Example 
% A set of radiobuttons with the strings 'bttn1', 'bttn2', 'bttn3' 
% each have the tag 'radiogroup1'. Clicking on 'bttn2' will set 
% its 'Value' property to 1 and that of the others to 0. 
% The 'userdata' of all three will have the 2 fields: 
% 'tag' = 'radiogroup1' 
% 'active' = 'bttn2' 
% 
%    The 'active' field can be retrieved with GETRADIOGROUP 
% 
% U. Egert 6/98 
% 
% an improvement would be to list handles to the members of the 
% group in the userdata of each member. This would speed up the 
% switching and allow individual 'tag' strings. 
% 
% See also GETRADIOGROUP
