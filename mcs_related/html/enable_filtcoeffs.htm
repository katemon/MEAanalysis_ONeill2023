Function synopsis

ENABLE_FILTERCOEFFS - adjusts filter coefficient fields in the sweep statistics box

Function comments

 
% callback function for the popup list with 
% filtertypes used in swp_statisticsbox.m 
% <filtertype> is an index into the list shown in 
% the filtertype popup menu. It enables/disables the 
% filter coefficient edit fields suitable for the 
% following list of filter types. 
% 
% type         value     coeff1    coeff2    usage 
% no filter    1         off       off       none 
% moving avg.  2         on        off       coeff1 should give the window size 
% Sav. Golay   3         on        off       coeff1 should give savgol parameters or filter coeffs 
% other        4         on        on        both fields should give filter coeffs 
% 
% U. Egert 6/98
