Function synopsis

COUNTCELL - count the number of elements in a cell

Function comments

 
% N = COUNTCELL(C) returns the total number of non-cell elements of C. 
%  If any element itself is a cell this is resolved and included in the 
%  count. 'double' elements are add with prod(size(C)), 'struct' or 'char' 
% elements count 1 each. 
% 
% Example: 
% A = {[2x2 double],    'test'}; 
% N = countcell(A) will return N = 5; 
%
