removeDrift_IIR - remove slow signal drifts

Function comments

 
% dDuV_all = removeDrift_IIR(DuV_all, SSsv), where <DuV_all> is a matrix with 
% one dataset per row and <SSsv> is a 2-element vector indicating the 
% parameters for a low pass filter (IIR), removes 
% slow signal  drifts by subtracting an extremely smoothed copy of signal 
% from the  raw signal. To minimize memory usage this is done in a loop for 
% each  electrode separately. To allow for corrections of time-shifts by 
% the filter an intermediate step is used in which a lowpass filtered 
% signal is generated and then subtracted from the raw data. 
% 
% (c) U.Egert 2002
