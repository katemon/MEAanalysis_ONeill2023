Function synopsis

GLOBALIZE - makes existing variables global in a workspace.

Function comments

 
% GLOBALIZE(space,name1, name2, ...) makes variable <name> global in workspace <space>. 
% <name> must be a string, <space> must be 'caller' or 'base', both as 
% strings. GLOBALIZE looks for <name> in <space>, makes/creates it global 
% as necessary. The value of <name> is not changed. 
% 
% U. Egert 11/97 
%
