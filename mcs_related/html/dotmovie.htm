Function comments

 
%              certain binwidth 
% dotmovie(data, startwindow, binwidth, nbins, switchoff, varargin); 
% starts a movie where a series of events on a MEA electrode is shown 
% as a series of dots highlighting the corresponding recording positions. 
% <data> is an MxN matrix, with one row per electrode. <binwidth> is the 
% width of the bin evaluated for events. <nbins> the number of bins to 
% show. <switchoff> is a vector of channel IDs to electrode that should 
% not be shown even though they were recorded. 
% 
%
