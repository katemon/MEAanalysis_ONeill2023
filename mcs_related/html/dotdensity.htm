dotdensity - create a density plot from a dotplot

Function comments

 
% sh = dotdensity(lh, varargin) creates a pseudocolor 
% surface plot representing the distribution of values 
% in the xdata and ydata of line object <lh, if <lh> 
% is an object handle. If <lh> is a 2-column matrix 
% this will be used as a database. 
% 
% Note that the calculation is very slow with Matlab 
% 5.2 because hist.m is used. Matlab 5.3 uses histc 
% which is compiled in a dll and very much faster. 
% 
% Parameter/value pairs: 
% ..., 'ndensbins', n, ...    define the number of bins (default 100); 
% 
% U. Egert 2/2000 
% 
% See also BAR3 BAR3H HISTC
