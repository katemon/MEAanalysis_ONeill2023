Function synopsis

PPF_ANALYSISDLG1A - average across sets of columns in 2 and 3D matrices

Function comments

 
% [rmat, q] = PPF_ANALYSISDLG1A(inmat) opens a dialog box requesting 
% the entries necessary to calculate the median across several 
% columns (e.g. sweeps) in a matrix generated by the 'Display LFP 
% Features' function in the MEA-Tools menu. This function is useful 
% to plot, for example, stimulus/response relations. 
% 
% If multiple trigger events were defined for each sweep, the ratio 
% between 2 values along the 3rd dimension of the input matrix, 
% i.e. its planes, may be calculated by indicating the corresponding 
% trigger indices. Trigger indices point to the 3rd dimension of 
% the data matrix. The result is stored in the matrix indicated 
% in the corresponding line of the dialog box. 
% 
% Example: outmat be a 64x25x1 matrix holding the minima of the 
% LFP-responses to 25 stimuli. Within this matrix each set of 5 
% stimuli had the same conditions (1:5, 6:10, etc.). ppf_analysisdlg1 
% will calculate the average across these 5 sweeps and plot the 
% result against the xvalues indicated via the dialog box. 
% 
% Note that this works only for equal number of repetitions per 
% stimulus condition. Columns exceeding full multiples of the repetitions 
% setting will be ignored. <q> gives the number of trials ignored. 
% 
% 
% 
% (c) U. Egert 1999 
% See also PPF_ANALYSISDLG2
