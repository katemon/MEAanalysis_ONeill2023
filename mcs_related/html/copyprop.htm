Function synopsis

COPYPROP - copies a gcbo-property to a property of another object

Function comments

 
% copyprop(infield, outfield, targeth) copies the value of the property <infield> 
% of the current callback object to property <outfield> of the object 
% with the handle <targeth>. Generally <infield> and <outfield> must be 
% cell arrays with one-for-one source and target fields. 
% Alternative source objects can be specified by their handle with 
% the parameter value pair 'sourceobj', <sourcehandle>. 
% 
% If <infield> is a cell array and <outfield> is 'userdata' (string) the value of 
% the <infield> properties are copied to the target 'userdata' field as 
% structure fields. 
% 
% U. Egert 6/98
