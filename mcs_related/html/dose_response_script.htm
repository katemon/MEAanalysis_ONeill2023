%############################################################################ 
% 
%                        DOSE/RESPONSE EVALUATION 
%  Contents: 
%  - Dose response evaluations: 
%       Steps: 1. adjust configuration 
%              2. define and describe epoch limits 
%              3. evaluate them (e.g. calculate a mean for each epoch) 
%                 and create a dose response plot 
% 
%  - Utilities 
% 
%  U. Egert 11/98 
% 
% This script will assist you in defining time windows (epochs) representing 
% specific experimental conditions based on the graphical representation of 
% the value of a parameter of your choice vs. time. Example: LFP area vs. time 
% while a test substance was applied in a number of concentrations. 
% 
% The script works on datasets created by either the 'LFP-Analysis I' & 'Evaluate LFP 
% Features', or the 'Spike Rate Analysis' functions in the MEATools suite. 
% 
%############################################################################
