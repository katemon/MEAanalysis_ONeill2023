Function synopsis

GETNAME - finds the MEA-electrode name for a channel ID

Function comments

 
% [namestr, padnummatrix] = getname(cid) retrieves the name <namestr>of a MEA electrode 
% defined by its channelid <cid>. The class of <namestr> can be 
% defined by a parameter/value pair: 
% 'returnclass'     'mat' will return a string matrix (default) 
% 'returnclass'     'cell' will return a cell array of strings 
% 
% <padnummatrix> is a matrix giving the column/row electrode identifiers (names) 
% 
% globals MEAINFO RACKSETUP 
%
