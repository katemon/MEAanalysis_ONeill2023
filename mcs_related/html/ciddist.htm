Function synopsis

ciddist - calculate the distance betw. 2 matrix pos.

Function comments

 
% dist = ciddist(cid1, cid2, Ms, gridw) returns the spatial 
% distances between 2 positions defined by the linear indices 
% <cid1>, <cid2> in a matrix of size <Ms> and gridwidth 
% <gridw>. <gridw> must be a 2 element vector giving the row 
% and column spacing. If <cid1> is a vector <cid2> must 
% have the same size and <dist> will be the pairwise 
% distances.
