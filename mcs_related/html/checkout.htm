checkout - check out the current meatools version to a new directory

Function comments

 
% checkout(newdir, pathonly) - copies the current meatools version, i.e. the 
% files in the meatools and meatools/mcintfac directories to a new directory. 
% If <newdir> is omitted or empty <newdir> will be defined from the current date, 
% e.g. if today is 05-Apr-2000 <newdir> will be 'meatools05-Apr-2000' in the 
% path defined by mfilepath.m. 
% 
% All files in the meatools directory and the meatools/mcintfac directory 
% will the be copied to <newdir>. These directories will be appended to the 
% matlab searchpath while the original meatools and meatools/mcintfac directories 
% will be removed from the search path. Finally, all functions will be cleared. 
% In case you have an active datastream object in your workspace you will have 
% to redefine it. 
% 
% (c) U. Egert 4/2000
