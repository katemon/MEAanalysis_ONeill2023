cfgstruct2tag - assign values from a structure to object properties

Function comments

 
% a = cfgstruct2tag(fh, cfg, varargin) 
% works through the following sequence 
% - find the objects with tagfields corresponding to the fieldnames of 
%     the structure <cfg> 
% - assign the respective structure field values to the 'userdata' property 
%     of these objects 
% - return the object handles 
% 
% The targetfield is 'userdata' by default and may be modified by 
% a parameter/value pair with the parameter 'targetfield'. 
% 
% U. Egert
