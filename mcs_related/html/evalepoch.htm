evalepoch - evaluate a sequence of data ranges

Function comments

 
% [ep_eval, ep_std] = evalepoch(epoch_data, sourcedata, varargin) 
% 
% IN 
% <epoch_data>      cell array defining stimulus type, stimulus value, start and stop time 
% <sourcedata>      matrix or cell array of vectors with the data to evaluate, e.g. spikerates; 
%                   It is assumed that the entries correspond to equidistant samples. 
% 
% <varargin> modifies default setting with parameter/value pairs: 
% 
%    'dim'          dimension along which to evaluate a matrix, default is 2, i.e. columns. 
%    'ftype'        function to use for evaluation, e.g. 'mean' 
%    'normalize'    when set to 1 normalizes the epoch values with respect to the 
%                        norm_refepoch. Default is 0. 
%    'norm_refepoch' index identifying the reference epoch (i.e. row) in <epoch_data> 
% 
% OUT 
% <ep_eval>         results matrix with one column for each dataset and one row for each epoch. 
%                   Columns are arranged as in sourcedata, rows in the sequence they were selected. 
% <ep_st>           corresponding matrix with standard deviation (standard error when normalized) 
% 
% U. Egert 10/98
