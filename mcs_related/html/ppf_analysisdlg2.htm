ppf_analysisdlg2 - dialog to reshape 'outmat' matrices

Function comments

 
% ppf_analysisdlg2 reshapes 'outmat' matrices generated by 
% the 'Display LFP features' function of the MEA-Tools. 
% 
% ppf_analysisdlg2 is useful to rearrange data in which 
% several successive short sweeps with one trigger event each 
% should be interpreted like a train of trigger events within 
% one long sweep. ppf_analysisdlg2 creates a matrix that corresponds 
% to the result of the 'evaluate LFP features' in the MEA-Tools 
% menu of such a multi-trigger sweep. 
% 
% example 
% A be a 64xNx1 matrix such that 2 successive columns 
% describe the response to a paired pulse stimulus. 
% ppf_analysisdlg2 will return a 64xN/2x2 matrix. 
% 
% (c) U. Egert 7/99 
% 
% See also PPF_ANALYSIS PPF_ANALYSISDLG1
