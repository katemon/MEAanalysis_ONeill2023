set2userdata - copy variables to the userdata of a graphics object

Function comments

 
% set2userdata(h, varargin) 
% places the content of the variables in <varargin> (must not strings!) 
% as fields with the variables name in the userdata property of 
% the object with the graphics handle <h>. 
% 
% Note that the input variables can not be the results of a calculation 
% or adressing command, e.g. a(5). 
% 
% (c) 2002 U. Egert
