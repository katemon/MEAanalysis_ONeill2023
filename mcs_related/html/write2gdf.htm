Function synopsis

WRITE2GDF - converts channel/time vectors to gdf

Function comments

 
% gdfmat = write2gdf(C,T, varargin) converts column vectors giving the ChannelID 
% and the spiketime to a N x 2 matrix giving ChannelID in the 1st and 
% event times in the 2nd column. 
% 
% Default parameters can be modified by parameter/value pairs: 
% 
%  'fname' ('none')        the target file 
%  'idx' (all)             a vector of indices into <C> and <T> to define 
%                          the export values 
%  'intimeunits' (1e-6)    time units of <T> given as fractions of seconds 
%  'outtimeunits'(1e-6)    time units to use for the gdf file 
% 
% 
% 
% A string added will be interpreted as a path/filename and writemcs2gdf 
% will attempt to store the resulting matrix in ASCII code in this file. 
% 
% These options may be combined in any sequence. 
% 
% Example: 
% write2gdf(C,T, 'idx', 1:1000, 'outtimeunits', '1e-3', 'fname', 'c:\temp\test.gdf') 
% will store a 1000x2 matrix with time in msec multiples in the file 
% 'c:\temp\test.gdf'. 
% 
% See also MCS2GDF 
% 
% U. Egert 5/98
