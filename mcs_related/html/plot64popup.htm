Function synopsis

plot64popup - copies an axis in a plot64.m figure to a new window (MEATools comp.)

Function comments

 
% plot64popup is callback function used to 'popup' an axis in the 8x8 
% subplot figures created by plot64.m. 'Left-click' creates a new figure 
% for the axis that was clicked. 'Shift-left-click' copies the traces into 
% the last figure of this kind that was activated (irrespective of that 
% figures original parent). The 'new' trace will be recolored and a legend 
% will be added to the figure. 'Ctrl-click' or 'right-click' will delete 
% the subplot. See 'plot' function for color sequence and line types. 
% 
% (c) U. Egert 2/99 
% 
% See also PLOT64 PLOT64GUI
