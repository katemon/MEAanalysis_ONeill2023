Function synopsis

GUIcfg2struc - converts GUI-configuration structure to a parameter/value structure

Function comments

 
% pvp = GUIcfg2struc(cfg, switchvals, varargin) uses the fieldnames of <cfg> 
% as parameters and takes the 'string' and 'value' fields of these subfields to build 
% a structure holding parameters as fieldnames with the appropriate values. 
% 
% If a parameter/value pair 'usefields', <fieldnamelist> is given, only these 
% fields are used. 
% 
% <cfg> must comply to the structure style described in setguiud.m. 
% The optional cell argument <switchvals> determines how the checkbox/radiobutton 
% values 0 and 1 should be translated. <switchvals{1}> will be returned for 0, 
% <switchvals{2}> will be returned for 1. 
% Default setting is {'no', 'yes'}. 
% 
% U. Egert 6/98 
% 
% See also GETGUIUD SETGUIUD CFG2BASE CFG2FIGUD SETSELF CELLFUNC_CALL
