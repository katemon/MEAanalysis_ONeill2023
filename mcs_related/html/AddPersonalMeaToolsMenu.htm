Function comments

 
% AddPersonalMeaToolsMenu is a utility to help you to personalize the 
% main menu of the MEA-Tools. You can add items to each of the main headings 
% by modifying this file. These variables hold the graphic handles for the 
% main menu items: 
% 
% STh  -'Spike Tools' 
% LFPTh - 'LFP Tools' 
% CDTh - 'Continuous Data Tools' 
% SDTh - 'Suppl. Data Tools' 
% DFTh - 'Display Functions' 
% Doch - 'Documentation' 
% 
% The menu items will be added to the MEA-Tools main page for this session 
% only. To permanently adjust the menu you might call AddPersonalMeaToolsMenu 
% from the meatools.m file.
