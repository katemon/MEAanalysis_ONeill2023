CSLIDER - creates a colour scale slider

Function comments

 
% h = cslider(cd, varargin) creates a colour scale slider in 
% figure. <cd> are the color values used in the figure.The 
% target and other properties must be set with parameter value 
% pairs. The function of the slider can be switched between 
% setting the zero position of the color range and adjusting the 
% color range by checking the context menu called with the 
% right mouse button on the figure. 
% 
% 'targetax'      the axis on which the slider should work (default gca) 
% 'targetfig'     the figure to which the slider is added (default gcf) 
% 'pos'           the sliders position (default is based on the 
%                    targetax position) 
% 'step'            stepsize of the slider (default [0.02 0.1]) 
% 'cmin'          default <mind> 
% 'dmm'           datarange for the colorbar 
% ' 
% 
% (c) U. Egert 9/98 
% 
% See also SETCSLIDE
