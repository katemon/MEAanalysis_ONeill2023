Function synopsis

MEAMOVIE - displays a 64 x N-matrix as an animated pseudocolor plot [MEA/ab]

Function comments

 
%   MEAMOVIE(MAT) where <MAT> is a 64 x N Matrix displays the values of <MAT> 
%   columnwise as interpolated 8 x 8 pseudocolor frames. 
% 
%   The following parameter/value-pairs can be given, separated by commas: 
%     frames:       range of frames (columns) out of data that are to be 
%                   displayed 
%     badPads:      indices of pads whose data are to be discarded 
%     skip:         number of frames that are to be skipped between each 
%                   two that are displayed 
%     zlim:         minimum and maximum of the color- and z-axis in µV 
%     ipFactor:     number of rounds of interpolation 
%     ipMethod:     method of interpolation 
%                   [ 'linear', 'nearest', {'cubic'}, 'spline' ] 
%     viewDim:      start the movie as 2- or 3-dimensional surface plot 
%                   [ {'2D'}, '3D'] 
%     loop:         restart the movie automatically [ {'on'}, 'off' ] 
%     preIp:        batch interpolation [ 'on', {'off'} ] 
%     ctrl:         show run/stop/continue-button [ {'on'}, 'off' ] 
%     shadingIp:    shading interpolated [ {'on'}, 'off' ] 
%     markers:      markers to be plotted 
%     markerStyle:  style of these markers (as matlab plotting style string) 
%     saveas:       path and group name for jpeg-files, e.g. 'saveas', 'c:\temp\test' 
% 
%  See also: STARTMOVIE, STARTMOVIEGUI
