Function synopsis

SETGUIUD - sets fields in a GUI as given in a structure

Function comments

 
% 
% setGUIud(CFG, fh) searches figure <fh> for objects 
% with the 'tag' the same as the fieldnames of <CFG>. 
% For these objects the properties given by the subfieldnames 
% are set to the subfield values. The subfields are written 
% to the 'userdata' field of these objects. Existing 'userdata' 
% are overwritten unless they are structures! 
% 
% setGUIud restores structures generated by GETGUIUD to the 
% GUI <fh>. If <fh> is omitted the current callback figure is 
% used. 
% 
% Example 
% CFG.PopUpMenu1.value = 1; 
% setGUIud(CFG) will set the 'value' property of the object with 
% its 'tag' = 'PopUpMenu1' to 1. 
% 
% 
% Known bugs 
% SETGUIUD will not restore the settings of a radiobutton group 
% as defined by SETRADIOGROUP. 
% 
% U. Egert 6/98 
% 
% See also GETGUIUD CFG2BASE CFG2FIGUD
