pc_vs_time - show PC scaling factors vs. experimental time

Function comments

 
% pc_vs_time(electrode, CN, PC, varargin) show PC scaling factors 
% vs. experimental time. Electrode is a string defining an electrode, 
% CN the requested cluster number, PC an index into the 
% list of mean spike & principle components. 
% 
% To detect trends in spike shapes 
% this tool allows you to plot the principle component 
% contributions in a particular spike cluster vs. time. 
% 
% The contributions vector is filter with a moving 
% average window. Modify by 
% ..., 'fparams', N) where N is the width of 
% moving average window given in no. of spikes (which 
% is a quick hack) 
% 
% (c) U. Egert 2000
