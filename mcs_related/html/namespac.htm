Function synopsis

namespac - define members of struct name in caller's workspace

Function comments

 
% *************************************************************** 
% *                                                             * 
% * Usage:                                                      * 
% *       namespace('name', 'fname') is typically called with 	 * 
% *       a struct name 													 * 
% *       declared global in the caller's workspace.            * 
% *       namespace('name') defines all members of struct       * 
% *       name in the caller's workspace.                       * 
% *       namespace('name', 'fname') uses only the fields 'fname'* 
% *       namespace() displays a warning if it is overriding    * 
% *       an already defined variable.                          * 
% *       namespace('name', 'fieldname') will extract only      * 
% *		 the fieldname specified. fieldname must be a cell     * 
% *       array of fieldnames.                                  * 
% *                                                             * 
% *                                                             * 
% * Example:                                                    * 
% *         Let x with x.y==5 be a global struct.               * 
% *         function f                                          * 
% *          global x; namespace(x);                            * 
% *          disp(y);                                           * 
% *         a matlab function.                                  * 
% *         A call to f will print 5.                           * 
% *                                                             * 
% * Future:                                                     * 
% *         write the reverse function endnamespace(name)       * 
% *                                                             * 
% * Uses:                                                       * 
% *       - evalin   (matlab5)                                  * 
% *       - assignin (matlab5)                                  * 
% *       - warning (matlab5)                                   * 
% *		                                                       * 
% * History:					                                		* 
% *         (1) second version                                  * 
% *         	 option to extract a selected fields added       *
