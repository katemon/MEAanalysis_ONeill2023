Function synopsis

SUBLABL - adds a text label to a subplot

Function comments

 
% comlabels = sublabl(handlemat, textmat, positionmat) labels the subplot 
% in the targetfigure with the text in <textmat>. Positions are defined 
% as described below and entered via <positionmat>. Target subplot 
% are defined by the handles in <handlemat>. A vector of handles 
% to the labels is returned in <comlabels>. 
% 
% Fontsize and fontname can be changed with the usual parameter/ 
% value pairs. 
% 
% Note that the default settings work with linear axes only 
% format für positionmat: 
% 
%		Pos.	correction factors 
%	  x	y 
% top left     11	1	1	% factor = 1 if default 
% top rigth    12	1	1	% should be used 
% bottom left  21	1	1 
% bottom right 22	1	1 
% center left  31	1	1 
% center right 32	1	1 
% 
% Example: 
% For an entry at the top left of a subplot <positionmat> would be 
% [11 1 1] or [11].
