Function synopsis

strtok2cell - parse a string into cellstr

Function comments

 
% t = strtok2cell(s,d, orientation) with <s> being 
% a string and <d> 
% a delimiter string <t> is a cell array of strings 
% that were separated by <d> in <s>. 
% 
% In any case, leading an trailing blanks will be removed. 
% 
% For string matrices <s> can be resolved along 
% 'rows' or 'column' by giving an optional 3rd 
% argument. 
% 
% <s> may also be a cell array of strings. 
% 
% (c) M. Diesmann/U. Egert 3/99 
% 
% See also STRTOK CELLSTR
