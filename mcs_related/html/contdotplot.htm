Function synopsis

contdotplot - simple dot-plot for spike times from MCRack files

Function comments

 
% contdotplot(electrodes, timerange) 
% creates a dot plot of spike times based on data created by parsing 
% MCRack files with spike cutouts. <electrodes> must be a cell array of 
% strings indicating which electrodes show be shown. If <electrodes> is 
% an empty cell array, the data for all electrodes will be used. 
% <timerange> must be a 2-element vector indicating the start and stop 
% times of the display window given in seconds. Two sliders can be used 
% to scroll along the x- and y-axes when the plot is zoomed, e.g. on 
% a smaller time window. 
% 
% (c) U. Egert 2002
