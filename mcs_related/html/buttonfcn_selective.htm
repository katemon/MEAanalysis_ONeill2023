Function synopsis

BUTTONFCN_SELECTIVE - evaluate a function depending on the mouse button used

Function comments

 
% buttonfcn_selective(leftfcn, middlefcn, rightfcn)is used to evaluate 
% a valid string depending the mouse button used to call buttonfcn_selective.m, 
% i.e. it is useful to design 'Callback', 'WindowButtonFcn', 'ButtonDownFcn', and 
% the like. It will evaluate leftfcn, middlefcn or rightfcn, so setting the 
% quotes in the strings may become somewhat complicated. One easy way to do 
% so is to write a normal valid callback and store that string as a 'userdata' 
% value. 
% 
% example (equiv. 'axis off' on left-click): 
% leftfcn = 'set(gca, ''visible'', ''off'')'; 
% figure 
% set(gcf, 'userdata', leftfcn); 
% plot(rand(1,100)) 
% set(gca, 'buttondownfcn', 'buttonfcn_selective(get(gcf, ''userdata''), [], [])'); 
% 
% 
% U. Egert 1/1999
