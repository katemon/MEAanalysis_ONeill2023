Function synopsis

GUIcfg2pvp - converts GUI-configuration struction to a parameter/value string

Function comments

 
% pvp = GUIcfg2pvp(cfg) uses the fieldnames of <cfg> as parameters and 
% takes the 'string' and 'value' fields of these subfields to build 
% a string that is suitable as parameter/value pair list when evaluated. 
% <cfg> must comply to the structure style described in setguiud.m. 
% The optional cell argument <switchval> determines how the checkbox/radiobutton 
% values 0 and 1 should be translated. <switchval{1}> will be returned for 0. 
% Default setting is {'no', 'yes'}. 
% 
% U. Egert 6/98 
% 
% See also GETGUIUD SETGUIUD CFG2BASE CFG2FIGUD SETSELF CELLFUNC_CALL
