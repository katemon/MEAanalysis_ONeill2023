LOWPASS - fast lowpass IIR filter

Function comments

 
% Version 1.0 - 17/09/04 - stefan.rotter@biologie.uni-freiburg.de 
% 
% y = lowpass(h,a,x) gives a filtered version of the time series x. If the 
% signal processing toolbox is available, a second-order butterworth 
% lowpass filter with cut-off frequency <a> in Hz is used. Othwerwise a 
% first-order exponential lowpass filter with "time constant" <a>. The 
% sampling PERIOD (!) is given by <h> in milliseconds, hence the sampling 
% frequency is assumed as 1/<h>. 
% 
% (c) Stefan Rotter/Ulrich Egert 2004
