Function synopsis

MCSDATE - recovers the recording date of a MCRacks-file

Function comments

 
% realdate = mcsdate(mcsd) converts the file record date 
% from the format used in MCRacks-files to date strings. 
% If no input argument is given it uses RACKSETUP.yrecdate 
% as constructed with the command <RACKSETUP.yrecdate = 
% 1970 + dword/3600/24/365> in the file header parsing 
% program pshdspx82.m to pshdspx85.m. 
% If <mcsd> is a datastream object the recording start time will be 
% determined from the field 'recording date' and converted into a date 
% string. 
% Alternatively the old date version (< v. 1.44) can be given as input 
% argument and will be converted. 
% 
% 
% See also DATESTR DATASTRM 
% U. Egert 2/98
