Function synopsis

cbsag_smallmoviefig - creates a cerebellum-LFP movie demo

Function comments

 
% cbsag_smallmoviefig loads a micrograph of an acute cerebellum slice 
% and displays a movie of a stimulus induced local field potential in 
% that slice. The dataset used to generate the movie was cb090998b4cont.001 
% from the slice shown in Cb090998_e35b.jpg. 
% 
% The data shown are an average of the sweeps 1:5, filtered with 
% savgol(25,3,0) and clear from the stimulus artifact with a template 
% generated from the trace of electrode # 17, i.e. the one most distant 
% from the stimulus site. 
% 
% (c) U. Egert 2/99
