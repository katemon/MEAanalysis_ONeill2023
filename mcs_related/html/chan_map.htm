Function synopsis

CHAN_MAP - plots a MEA grid with electrode positions

Function comments

 
% [idxfill_pads,idxbad_pads]=chan_map(idxexist_pads, identify) 
% will plot a square with electrode positions marked as defined by 
% <idxexist_pads> and parameter/value pairs. <idxexist_pads> must 
% be linear indices into an 8x8 matrix and requires meainfo.mat to 
% work. 
% 
% parameter/value definitions 
% 
% 'identify' (0)        set to 1 to identify indices for electrode positions (ChannelID) 
%                       by mouse clicks. ChannelIDs will be returned in <idxfill_pads>. 
%                       The remaining electrode positions will be in <idxbad_pads>. The selection 
%                       is ended by clicking the right mouse button. 
% 'existmarker' ('bo')  marker an color for electrode positions in <idxexist_pads> 
% 'badmarker' ('r*')    <idxbad_pads> marker 
% 'anymarker' ('go')    marker for all positions in the grid 
% 'figname' ('MEA-Map') name of the map figure 
% 'plotoption' ('all')  any of 
%                          'all' - which will show all electrode + <idxexist_pads> 
%                          'exist' -  which will show <idxexist_pads> only 
%                          'bad' - which will show electrodes not covered by <idxexist_pads> 
% 
% See helpwin plot for valid marker styles. 
% 
% U. Egert 2/99 
% 
% See als PLOT MEAINFO
