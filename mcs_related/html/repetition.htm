Function comments

 
% w=repetition(v,n,l,o), repeat segments of length l n times 
% *************************************************************** 
% *                                                             * 
% * Usage:                                                      * 
% *           v, row or column vector                           * 
% *           n, number of times segments should be repeated    * 
% *           l, length of segments                             * 
% *              if l is ommited all v is taken as one segment  * 
% *              (l=length(v).                                  * 
% *           w, result with orientation of v                   * 
% *                                                             * 
% * Note:                                                       * 
% *  In the degenerate case length(v)==1 the row or column      * 
% *  vector v reduces to a scalar. However, w is defined to     * 
% *  have the same orientation as v. For scalar v repetition    * 
% *  needs the property string 'row' or 'col' as the last       * 
% *  argument, otherwise a                                      * 
% *              'Repetition:ScalarNotOriented'                 * 
% *  error is raised. Applications using repetition in an       * 
% *  environment where v may reduce to a scalar should always   * 
% *  use a version of repetition with forced orientation:       * 
% *     e.g.  repetition(v,2,'row') .                           * 
% *  This is essential because the index operator() has         * 
% *  different definitions for scalars and vectors:             * 
% *                                                             * 
% *      v is   ! orientation of v(i)                           * 
% *    -------------------------------------------------        * 
% *     scalar  ! matrix with orientation of i                  * 
% *     vector  ! vector with orientation of v                  * 
% *     matrix  ! matrix with orientation of i                  * 
% *                                                             * 
% * The orientation of a vector can be obtained by a call       * 
% * to orientation(v). orientation() returns a property string. * 
% *                                                             * 
% * Uses:                                                       * 
% *           part, orientation, copy                           * 
% *                                                             * 
% *                                                             * 
% * History:                                                    * 
% *         (4) repetrow and repetcol introduced, compiled      * 
% *             versions exist                                  * 
% *            MD, 1.4.1997, Freiburg                           * 
% *         (3) error checking simplified, now allows forced    * 
% *             orientation for l~=length(v) also.              * 
% *            MD, 16.2.1997, Jerusalem                         * 
% *         (2) orientation strings 'row', 'col' introduced to  * 
% *             safely handle degenerate case scalar(v)==1.     * 
% *            MD, 5.2.1997, Freiburg                           * 
% *         (1) call of partition removed                       * 
% *             The first version used partition for the one    * 
% *             segment case. The new version allows us to use  * 
% *             repetition in partition. Old warning:           * 
% *                "!!!NOTE, copy is implemented by the         * 
% *                 (v,n) version of repetition!!!" removed.    * 
% *            MD, 4.2.1997, Freiburg                           * 
% *         (0) first Version                                   * 
% *            MD, SG 29.11.1996, Jerusalem                     * 
% *                                                             * 
% *                                                             *

% *                                                             * 
% ***************************************************************
