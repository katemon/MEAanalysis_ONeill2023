Function synopsis

interpxval - find x-values when using interp2 on 2D matrices

Function comments

 
% xout=interpxval(xr,ipf) where <xr> is a 2-element vector and <ipf> an integer 
% returns the vector <xout>. <xout>is suitable as x-values in surf/image plots 
% of a matrix interpolated with interp2(M,ipf). 
% 
% Example: 
% A be an MxM matrix where M=8; 
% ipf = 3; 
% B = interp2(A,ipf); 
% xout = interpxval([1 8], ipf); 
% [X,Y] = meshgrid(xout); 
% surf(X,Y,B) 
%
