% A = spikerecdispctrl(cmd, varargin) 
% 
% <cmd> on of the following strings 
% 'defsource   define the cutout data source 
% 'set'        open an input window to adjust display parameters 
% 'info'       update status field 
% 'nextframe'  retrieve & display the next set of cutouts, retains zoom state 
% 'previousframe'retrieve & display the previous set of cutouts, retains zoom state 
% 'Wshift'     shift the display within the frame (e.g. after zooming) 
% 'unzoom'     resets the axis limits to the one defined with 'set'
