CHKGLOBAL - test for global state of variables and globalizes

Function comments

 
% ok = chkglobal(lspace, varargin) tests if the variables in <varargin> 
% are global in the <lspace> and globalizes them if they exist as local 
% variables or creates them global if the don't exist at all. 
% 
% The return value <ok> is 1 if the var. were found global, 0 if not, -1 if 
% it didn't exist. 
% 
% U. Egert 5/98 
%
