evaldsource_entries - evalute GUI entries from data source fields

% [out, inputType]=evalDSource_Entries(entry) 
% is used to evaluate <entry> such that it will 
% - create a datastream object 'D' if <entry> is the filename 
%   of an MCRack file (then <out>= [], inputType = 'filename') 
% - evaluate a string in the baseworkspace (then <out>= result, 
%   inputType = 'char') 
% - resolve cell entries as above 
% - return the result of numerical arguments, formulas or 
%   variables in <out>, inputType = 'var'. 
% - do nothing with datastream objects, inputType = 'datastream' 
% 
% evaldsource_entries is useful to evaluate GUI field that 
% ask for a datasource that can be one of the types above. 
% 
% (c) U. Egert 2001
