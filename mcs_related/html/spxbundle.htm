Function synopsis

SPXBUNDLE - finds and plots the traces crossing a time/voltage window

Function comments

 
% [Tpassh, idxt] = SPXBUNDLE returns the subsection 
% of the graphics handles (in the new plot) to the traces crossing a voltage window 
% and a time window which can be indicated with mouse clicks. 
% The return value <idxspx> gives the index to the spike events. 
% 
% SPXBUNDLE was written to work within a plot with multiple traces 
% as generated by 'parameterpltmany.m'. In these plots each trace 
% has its voltage values stored within the 'userdata' property of 
% the trace. The raw voltage data must be within a variable <ltraces>, 
% the filtered data in a variable <fitTRACES> in 'userdata'. SPXBUNDLE 
% uses either the filtered or raw data depending on the display type 
% used in the graphic. 
% 
% 
% U. Egert 2/98 
%####################################################################### 
% 
%                        PROGRAM 
% 
%#######################################################################
