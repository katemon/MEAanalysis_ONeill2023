Function synopsis

cid2subplotmap - creates subplots corresponding to MEA electrode positions

Function comments

 
% [sph, pos, cid2subidx] = cid2subplotmap(cid, meatyp, figureh) determines the electrode 
%  positions of channelID <cid> in a given MEA layout, creates these subplots 
% in figure <figureh> and returns subplot handles and positions. <cid2subidx> 
% is the index (not the handle) pointing to axes corresponding to cids in a subplot 
% figure, e.g. if 
% 
% cid = 32;        % corresponding to electrode # 48 in an 8x8 MEA, then 
% cid2subidx = 60; % points to the corresponding subplot with 
%                    subplot(8,8,60). 
% 
% MEA layouts supported (i.e. 'meatype' values): 
% 
%     '8x8'    60 electrodes on an 8x8 grid with corner positions not used 
% 
% (c) U. Egert 11/98
