Function synopsis

GETFILENAMES - returns a list of filenames in a path as cell array

Function comments

 
% [fnames, fpath] = getfilenames(dirpath, ext) returns a list of filenames in 
% <dirpath> in a cellstring. <ext> is an optional argument to filter for 
% file types. <ext> does not allow for wildcards. Using the string 'dir' as 
% extension argument will return a cell array with the names of direct 
% subdirectories to <dirpath> only. 
%
