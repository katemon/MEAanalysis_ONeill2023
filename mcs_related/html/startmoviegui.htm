Function synopsis

STARTMOVIEGUI - a graphical user interface for 'meamovie' [MEA/ab]

Function comments

 
% STARTMOVIEGUI without input parameters opens a figure with controls 
%   for most of the options in 'meamovie' and the possibility to 
%   select a file (which has to contain data recorded with MCSRack) 
%   and a sweep (number) from this file. Alternatively, the name 
%   of a variable from the base workspace representing either a datastrm 
%   object or a matrix with data (1 row per electrode) can be given. 
%   In the latter case the setting of the field 'sweep' is meaningless 
%   and the datarange shown must be given as column numbers. With 
%   datastream objects or MCRack files only the stream giving 'Electrode 
%   Raw Data' will be shown, others will be ignored. Set 'sweep' 
%   to one for continuous recording and adjust the range shown as 
%   usual. 
% 
%   The preferred matrix format is 64 x N, though some other 
%   formats may be correctly recognized (see the help for 'startmovie' for 
%   additional information). 
% 
%   STARTMOVIEGUI(datasource) where datasource is either the filename of an 
%   MCSRacks sweep file or a variable name presets the 'datasource'-field 
%   of the gui with the contents of the paramater <datasource> 
% 
%   Additional help for each parameter is provided by the tooltips. Note that 
%   it is not possible to indicate the data range displayed if your datasource 
%   is a matrix. 
% 
%   See also MEAMOVIE, STARTMOVIE
