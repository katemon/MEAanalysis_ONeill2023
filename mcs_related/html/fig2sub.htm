Function synopsis

fig2sub - copies a full figure to a subplot position

Function comments

 
% fig2sub(s,t,newpos) copies all objects of figure <s> to 
% the subplot position <newpos> of figure(<t>). All objects 
% are resized as needed to fit to the new position. Note that 
% existing subplots are NOT replaced or deleted. 
% 
% Example 
% t = figure; s = figure; 
% plot(rand(1,100)); 
% fig2sub(s,t,[2,2,4]); - copies figure(s) to the lower left 
%    quadrant of figure(t). 
% 
% U. Egert 8/98
