% <data_nanidx> = measurfplot(<data>, <varargin>) 
% creates a surface plot from a MEA array which can be adjustet by 
% parameter/value pairs 
% 
% <data> should be a 64x1 or 68x1 array. 
% 
% parameter/value pairs with their default values 
% 
% 'cfg_idxbad_pads' []                -    ChannelIDs of electrodes (rows in <data> )with no or bad signals which 
%                                          should be interpolated 
% 'cfg_idxbad_nonip'[]                -    ChannelIDs of electrodes with no or bad signals which 
%                                          should be omitted 
% 'cfg_ipDepth'     2                 -    interpolation depth 
% 'cfg_ipMethod'    'cubic'           -    interpolation method 
% 'cfg_title'       'measurfplot'     -    title for surfaceplot 
% 'cfg_ystring'     'voltage [ \muV]' -    label for colorbar 
% 'cfg_zlim'        [min(data(:)) max(data(:))] -    limits for the colorbar 
% 'cfg_zlim_blank'  0                 -    Set to 1 if zvalues outside 'cfg_zlim' should be blanked 
% 'cfg_inv_cmap'    0                 -    Set to 1 if colormap should be inversed 
% 'cfg_cldata'      0                 -    switch to ask to clear certain values above/below threshold 
%                                       the following parameter/value pairs will then be given to 
%                                       setsurfval2nan: 
% ----------------------------- 
% 'theta'           value for the threshold, if theta=NaN a histogram of 
%                        the values will be shown for graphical selection of theta. Setting theta 
%                        to Inf and thetasign to -1 will show all data. 
% 'thetasign'       0    neg. thresholds will result in all suprathreshold values being set to NaN. Pos. 
%                        thresholds likewise for all subthreshold values 
%                   1    all values smaller than 'theta' will be NaN, 
%                  -1    all above will be NaN 
%                  2/-2  theta is one value for each electrode. It will be interpolated just 
%                        like the data and thus form a threshold surface. 
%                        The threshold matrix will be retrieved from the 
%                        userdata field of the surface plot created by 
%                        measurfplot. 
% 'refdata'         []   alternative dataset to evaluate for sub/supra-threshold entries 
%                        NOTE: this dataset needs to be exactly the size of the 
%                        zdata set of the target surface object. You may want to use 
%                        the interpolation function to create this set. 
% 'nanidx'          []   this parameter may be used to directly indicate which values 
%                        should be set to NaN. 
% 'autozlim'        0    determines the z-level and color limits depending on the threshold 
%                        and sign settings. Set to 1 to adjust these limits with 'cfg_zlim'. 
%                        'autozlim' is not used in this function but inherited to setsurfval2nan.m. 
% 'cfg_zlim'             limits for z-level and colorscale, determined as max and min of 
%                        <data> unless modified as a parameter value pair with 
%                        a two element vector as value indicating upper an lower limits. 
% ----------------------------- 
% <data_nanidx> indicates the zdata entries that were set to NaN, if 'cfg_cldata' = 1. 
% 
% See also SETSURFVAL2NAN
