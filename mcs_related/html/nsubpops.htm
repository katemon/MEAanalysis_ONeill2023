Function synopsis

NSUBPOPS - shows the subpopulations sorted for one or all electrode (MEATools)

Function comments

 
% [nsubpopmat, enamelist, subpoplist] = NSUBPOPS(ename) no. of spikes per subpopulation 
%              of electrode <ename>. Creates a graphical display of the result. 
% 
% <ename>      can be a either a string giving the electrode name or a number 
%              giving its ChannelID. If <ename> is omitted all subpopulations for 
%              all electrodes are listed. 
% <nspspxV>     is a 2 column vector giving the CLUSTERNUMBER assigned to a 
%              subpopulation (1st column) and the no. of spikes for that 
%              cluster (2nd column). Each row refers to a subpopulation. 
% 
% uses 
%	getidx 
% 
% global variables used: 
% 
%	CLUSTERNUMBER 
%    SOURCECHANNELNUMBER 
%    RACKSETUP 
% 
% See also GETSUBPOPS NSPX 
% 
% U.Egert 12/97
