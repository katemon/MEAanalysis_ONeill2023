Function synopsis

UNSORTALL - utility to reset subpopulation assignments.

Function comments

 
% UNSORTALL(en, sp) is restricted to setting the CLUSTERNUMBER of 
%                   either a given subpopulation <sp> on electrode <en> 
%                   to zero. 
% 
% <en>     if a string is given the corresponding ChannelID is determined and 
%          used. Alternatively the ChannelID can be given as a number. 
% 
% The PARSESETUP.Clusternames corresponding to the given subpopulation of spikes 
% is deleted. 
% 
% If <sp> is omitted or set to zero all CLUSTERNUMBERs of that electrode 
% are reset. 
% 
% If no input arguments are given all sorting results are reset by deleting 
% CLUSTERNUMBER and PARSESETUP.Clusternames. 
% 
% uses 
%    getidx 
%    getcid 
% 
% U. Egert 1/98
