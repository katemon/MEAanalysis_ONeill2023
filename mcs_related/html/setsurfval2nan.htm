setsurfval2nan - set sub/supra threshold values in a surface to NaN

Function comments

 
% nanidx=setsurfval2nan(surfobjh, varargin) 
% sets all z-values of the surface object <surfobjh> to NaN depending 
% on their position above or below a threshold. If only <surfobjh> 
% is given a threshold will be queried via mouse click based on a 
% histogram of the value distribution. By default, neg. thresholds 
% will result in all suprathreshold values being set to NaN. Pos. 
% thresholds likewise for all subthreshold values. Threshold 
% and direction can be modified manually with parameter/value pairs. 
% 
% <nanidx> indicates the zdata entries that were set to NaN. 
% 
% 'theta'           value for the threshold, if theta=NaN a histogram of 
%                        the values will be shown for graphical selection of theta. Setting theta 
%                        to Inf and thetasign to -1 will show all data. 
% 'thetasign'       0    for automatic direction as described above, 
%                   1    all values smaller than 'theta' will be NaN, 
%                  -1    all above will be NaN 
%                  -2    the threshold matrix will be retrieved from the 
%                        userdata field of the surface plot created by 
%                        measurfplot 
% 'refdata'         []   alternative dataset to evaluate for sub/supra-threshold entries 
%                        NOTE: this dataset needs to be exactly the size of the 
%                        zdata set of the target surface object. You may want to use 
%                        the interpolation function to create this set. 
% 'nanidx           []   this parameter may be used to directly indicate which values 
%`                       should be set to NaN. 
% 'autozlim'        0    determines the z-level and color limits depending on the threshold 
%                        and sign settings. Set to 1 to adjust these limits with 'cfg_zlim' 
% 'cfg_zlim'             limits for z-level and colorscale, inherited from the surface object 
%                        unless modified based on 'autozlim', and a parameter value pair with 
%                        a two element vector as value indicating upper an lower limits. 
% 
% To enter these parameters via a GUI call setsurfval2nan('gui') or 
% setsurfval2nan(<h>, 'gui') where <h> is a figure handle. 
% 
% See also MEASURFPLOT 
%
