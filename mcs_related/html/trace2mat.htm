Function synopsis

trace2mat - retrieve spike cutouts to a column matrix (MEATools)

Function comments

 
% 
% TRACES  = trace2mat(idx, tracetype) retrieves spike cutouts recorded 
% with MCRack to a matrix in the base workspace. Each column in <TRACES> 
% corresponds to one spike waveform. <idx> is a vector of 
% indices into the global variable <spkpointer>, <tracetype> can be 
% 'raw' or 'filtered' resulting in either unfiltered or Savitzky-Golay 
% filtered cutouts. Default filter coefficients [5,3] may be modified 
% by the parameter value pair 
% 
%           ..., svparams, [a,b]) 
% 
% If called without input arguments an input dialog box is opened requesting 
% the necessary information. 
% 
% If used with datastream objects it is necessary to use the global variable 
% <D> for the object. Other names are not allowed. 
% 
% To access spike of a specific channel ID with datastream object extend with 
% ..., 'dstreamobj', <name>, 'channelid', <cid>, ... 
% 
% (c) U. Egert 3/99 
% 
% See also GETTRACE
