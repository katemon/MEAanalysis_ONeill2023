Function comments

 
% ------------------------------------------------------------------------ 
% CBDEMO2_SCRIPT assists in creating a contour plot overlayed on a bitmap 
% image. The idea is to show voltage or time contours from MCRack data 
% on a bitmap image of a slice. 
% 
% Bitmap figure size is currently optimized for square image. The 
% contour axis needs to be resize by mouse action to fit the MEA grid. 
% Since you cannot rotate axes conveniently take care to orient the 
% bitmap when you scanning it. The contour plot axis is transparent, thus 
% the bitmap will show througth. 
% The last section of this script was an effort to produce a contour movie 
% running above the bitmap. Unfortunately, Matlab will not store the bitmap 
% values correctly, so the replay is a mess. Neither can you generate trans- 
% parent movie frames. 
% 
% (c) U. Egert 2/99 
% 
% 
% See also CHANMAP_ON_IMG IMG_SURF
