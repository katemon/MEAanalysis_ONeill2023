Function synopsis

CELLFUNC_CALL - calls a function with the arguments given as cell array

Function comments

 
% c = CELLFUNC_CALL(func, varargin) generates a command string from the 
%   input arguments. The function <func> is called with the content of 
%   <varargin> as separate input arguments. 
% 
% Example 
% the normal call to function FUNC be z=func(a, b, c) then 
% x = {a b c} would invoke that function with 
% z = CELLFUNC_CALL(func, x); 
% 
% U. Egert 6/98
