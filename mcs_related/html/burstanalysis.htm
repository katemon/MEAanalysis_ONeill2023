Function synopsis

burstanalysis - analyze spike trains for burst patterns (MEATools component)

Function comments

 
% 
% [nbursts,burstpos] = burstanalysis(threshold, spx_per_burst, cid, CN) 
% identifies burst events in a spike train. 
% Burst are identified as a series of short interspike intervals following a long 
% interspike interval. 
% 
% <threshold> gives the maximum interspike-intervall duration allow within a burst (ms). 
% <spx_per_burst> gives the minimal no. of spikes required for a burst. 
% <nbursts>    gives the no. of bursts, not the no. of spikes in bursts!!! 
% <burstpos>   a vector into the event series used pointing to the first spike in 
%              each burst 
% 
% Default settings can be modified with parameter/value pairs: 
% 'range'      a 2-element vector giving the start and end time of the data 
%              range to be processed, given in the same units as SPIKETIME uses. 
%              default: all spikes are used, 
% 'eventtimes' can be a vector of times that are analyzed for bursts. 
%              Def: times are selected from SPIKETIME based on <cid> and <CN> 
% 
% (c) U. Egert 10/98
