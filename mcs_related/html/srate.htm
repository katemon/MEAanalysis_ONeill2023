Function synopsis

SRATE - calculates local event rates with a moving window

Function comments

 
% [n, ravg, rstd, rmed, rN] = srate(events, twindow, shiftval, doplot) 
% 
% Input arguments: 
% events    usually event times 
% twindow   the time window across which the rate is calculated 
% shiftval  increments for moving the window 
% dotplot    creates a rate plot if set to 'yes' 
% scale     'frequency' or 'counts' (default) 
% 
% output arguments 
% n			a vector giving the counts per time unit (counts/twindow) 
% ravg		mean rate across the input vector 
% rstd		standard  across the input vector 
% rmed		median  across the input vector 
% rN			number of windows used 
% 
% U. Egert 12/97
