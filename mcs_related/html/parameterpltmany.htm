Function synopsis

PARAMETERPLTMANY - plots traces from the original MCS datafile

Function comments

 
% h = PARAMETERPLTMANY(dataset, datatype, varargin) 
% with varargin giving parameter/value pairs 
% 
% <dataset> can be a string giving an electrode name, a list of pointers to the onset 
% of the traces in the original data file,a ChannelID or indices into spkpointer. 
% IMPORTANT: the <datatype> used must be given as 'pointer', 'electrodename', 'cid', or 
% 'index'. 
% 
% Other options are 
% 'subpopulation' - referring to a subpopulation value created during spike sorting (integer). 
% 'tracetype' - can be 'raw' (default) or 'filtered', in which case Savitzky-Golay 
%               filtered traces will be displayed. 
% 'datafile' - path\filename to the raw MCS datafile 
% 
% <h> is a vector of handles to the traces. 
% 
% 
% U. Egert 1/98
