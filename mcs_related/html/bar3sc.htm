Function synopsis

bar3sc - 3D bar histogram with bar color corresponding to height

Function comments

 
% [bh]=bar3sc(D, varargin) 
% will create a 3D bar histogram as would bar3, except that the color 
% of each bar corresponds to its height. In this way you can view the 
% histogram from the top and still 'see' the entries. 
%
