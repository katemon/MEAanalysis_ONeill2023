LFP_EXTRACT - extract sweep characteristics in MCRack triggered sweeps (MEATools)

% [outp, fitidxrange] = lfp_extract(data, Trigidx, varargin) 
% Extracts the parameters below from triggered sweeps stored in MCRack datafiles. All 
% indices the columns in <data> 
% 
% contents of <outp> 
% 
% For  trigger event index <tx> <outp> holds for all columns (i.e. channels) of the data 
% segment: 
% outp(1,tx).polycoeff      are coefficients polynomial fit (double) 
% outp(1,tx).evalrangelim   are limits of the x-axis range fitted (2 doubles) 
% outp(1,tx).pminV          are the voltage-values with minima of the polynomial fit (cell) 
% outp(1,tx).pminTidx       are the indices of these values (cell) with respect to the index 
%                           of their respective triggers 
% outp(1,tx).pmaxV          are the voltage-values with maxima of the polynomial fit (cell) 
% outp(1,tx).pmaxTidx       are the indices of these values (cell) with respect to the index 
%                           of their respective triggers 
% outp(1,tx).dminV          are the minimal voltage-values of the traces (double) 
% outp(1,tx).dminTidx       are the indices to the minimal voltage-values of the traces (double) 
%                           with respect to the index of their respective triggers 
% outp(1,tx).dmaxV          are the maximal voltage-values of the traces (double) 
% outp(1,tx).dmaxTidx       are the indices to the maxmal voltage-values of the traces (double) 
%                           with respect to the index of their respective triggers 
% outp(1,tx).darea          is the sum of the abs. data values 
% 
% Thus, <out(1,tx).darea> will be a 1x64 vector giving the minimum value for each column 
% in the data 
% 
% U. Egert 6/98
