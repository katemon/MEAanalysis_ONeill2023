spxpar81 - extracts spike shape parameters (MEATools)

Function comments

 
% spike-parameter extraktion routine, Version 81 
% spxpar8 extracts some characteristic values from individual traces (intended for 
% spike waveform cutouts) and returns these (<PARMAT> and the corresponding indices 
% <parmatidx> into the traces as a row of values each. Traces are filtered with 
% a Savitzky-Golay filter (with 5, order 5) beforehand, the result of which is 
% returned in fitTRACES. 
% RACKSETUP must be a global variable containing the fields: 
%    RACKSETUP.ticks_pre_event 
%    RACKSETUP.MicrosecondsPerTick 
% 
% spxpar81 determines 
% vMin    the minimal voltage in the trace 
% vMax    the maximal voltage 
% tMin    the time of vMin 
% tMax    the time of vMax 
% tTWidth      the width of the peak between successive crossings of detection threshold 
% tMaxThreshld the time of the maximum value after crossing the threshold
