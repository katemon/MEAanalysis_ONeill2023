Function synopsis

getdir(cmd, startdir, outvarname) allows you to select a directory

Function comments

 
% from a hyperlinked list box. uigetfile does not provide this possibility. 
% Because matlab cannot identify the name of an output argument (as it does 
% for the name of an input argument, the selected directory name must be 
% stored in a variable in the base workspace. <cmd> is a switch statement. 
% For <cmd> == 'start' the GUI will open (same if started without arguments). 
% GETDIR can be started without arguments, in which case the target variable 
% will be 'gdir'. <startdir> optionally gives an initial directory, from 
% which to further select others. <outvarname> may be used to indicate 
% a different target variable. 
% 
% (c) U. Egert 2002
