dstreamsummary - writes a short datastream object summary to a 'Status' field

Function comments

 
% dstreamsummary(oname, streamname) summarizes datastream object information 
% in a figure. <oname> must be a string indicating the name of the datastream 
% object to use as a source. <streamname> is optional and points to the 
% datastream to evaluate. If your datastream object is named 'D' use 
% 'getfield(D, 'StreamNames') to see the available names. Unless <streamname> 
% is used the default is the first name in that list. 
% 
% See also SETSTATUS GETSTATUS
