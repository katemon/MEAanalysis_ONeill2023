Function synopsis

PLOTPRM8 - plot matrix columns against each other in subplots (MEATools)

Function comments

 
% PLOTPRM8(idxspx, varargin) plots values in the columns of a matrix (default is PARMAT) vs. each other. 
% <idxspx> must give the rows of the matrix to be used. Used in the spike sorting suite. Settings can 
% be modified with parameter/value pairs, none of which is required for default usage: 
% 
% parameter    legal values 
% 'dataset'    a matrix with rows per spike with the parameters in columns 
% 'holdit'     0 or 1, for plotting a second dataset on top of an existing plot 
% 'labels'     a cell array of strings with 1 string per column of <dataset> 
% 'flipxy'   1 or 0, exchanges x and y axes 
% 'useparams'  integer vector indicating the columns of <dataset> to be used 
% 'figname' any name of a sort-figure 
% 
% 
% global variables required: 
%     RACKSETUP (for default settings) 
%     PARSETUP (for default settings) 
%     PARMAT (for default settings) 
% 
% 
% U. Egert 12/1997 
% ****************************************************************************************************
