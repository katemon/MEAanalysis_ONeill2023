Function synopsis

scrollxaxis - adds a slider to move the x-axis window position.

Function comments

 
% scrollxaxis adds a small slider to a figure. In an unzoomed state 
% clicking on the slider will move the data left/right. If you zoomed 
% into the data, clicking on the slider will move x-axis limits of the 
% axis. Left-Click on the trough of the slider will move by one 
% full window, clicking on the arrow by 5% of the window size. 
% 
% A right-click with the mouse on the slider itself will restore the axis 
% to its auto-scale limits for that axis. 
% 
% The position of the window is limited by the max/min value in the whole 
% dataset. 
% 
% Known issues: the position of the slider will not be correct before the 
% first click after zooming. 
% 
% See also CREATESLIDER 
% 
% (c) U. Egert
