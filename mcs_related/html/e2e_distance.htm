Function synopsis

e2e_distance - calculates the direct distance between two MEA electrodes

Function comments

 
% d = e2e_distance(e, gridw,ref) were <e> is a cell array of string indicating 
% MEA electrodes (e.g. {'# 13', '# 15'}, the # sign is optional) calculates 
% the distance between these electrodes in units of <gridw>. <ref> 
% indicates the reference to which the distance is calculated. 
% Values can be 
% 0 (def), requesting the distance to e{1} 
% 1, requesting the distance between successive pairs 
% 2, requesting the distance for each pair. 
%
