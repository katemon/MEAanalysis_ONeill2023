SETFIELDLABEL - sets objectstrings according to the value of an object.

Function comments

 
% 
% setfieldlabel(tagstr, lstr, varargin) sets the string property of 
% an object with tag <tagstr> with a string drawn from the <lstr> 
% property of another object. Alternatively, if <lstr> is a cell 
% array the new string is drawn from that cell array. 
% By default the 'value' property of the current callback object 
% is used as index into the source string list generated from <lstr>. 
% Parameter-value pair 'uih', <objecthandle> will use the value of 
% the object with the handle <objecthandle>. 
% 
% 
% U. Egert 6/98
