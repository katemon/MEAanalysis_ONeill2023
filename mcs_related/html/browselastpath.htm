Function synopsis

BROWSELASTPATH - browse the last directory used

Function comments

 
% [dfile dpath] = BROWSELASTPATH(fmask) checks for a variable <lastdpath> in the 
% base workspace and uses the path indicated there to open a 
% 'load file'-GUI in that path. After selecting a file in this 
% or another directory it updates the <lastdpath> variable. The 
% file type can be filtered by giving a legal filename (with wild 
% cards) in <fmask>, e.g. *.mat. If <fmask> is omitted *.* is used. 
% 
% <dpath> is the selected path including a closen file separator, 
% <dfile> the selected files name. 
% 
% BROWSELASTPATH is especially useful when used as callback with a 
% browse button in a GUI. 
% 
% U. Egert 8/98
