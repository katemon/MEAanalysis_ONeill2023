Function synopsis

MOV2STRUCT - moves variables into a structure

Function comments

 
% structname = MOV2STRUCT(structname, inputvariables) moves the <inputvariables> 
%    into fields of <structname>. The <inputvariables> are cleared from the 
%    calling workspace. Input arguments must NOT be given as strings. 
% 
% example 
%    if: 
%    a = 1:5; b = 'asdf'; 
% 
%    then: 
%    c = mov2struct(c, a, b) 
%    will return the structure c with the field c.a and c.b. a and b will 
%    be deleted from the workspace as well as the global workspace. 
% 
% 
% 
% U.Egert 8/97 
% 
% See also MOV2STRUCT SETFIELD RMFIELD GETFIELD STRUCT
