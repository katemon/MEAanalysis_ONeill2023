POPGCO - pops a graphics object into a separate figure

Function comments

 
% targeth = POPGCO(varargin) can be used as callback or buttondownfcn 
% of line and axes objects. It copies the object full sized 
% into a new figure window. By default successive calls from 
% children of the same axis will copy these to the same 
% target axis. Use the parameter/value pair <'redrawstyle', 'new'> 
% to copy objects to individual figures. 
% 
% These default settings can be modified by parameter/value pairs: 
% targettype = 'figure'          'figure' or 'axes' 
% redrawstyle = 'new'            'new' for new figure, 'add' to add to a 
%                                previous figure 
% targeth = -1                   target figure handle for the object 
% targetname = ''                figure name for the new figure 
% clearbtnfcn = 1                1 to disable the popup function in the new 
%                                figure 
% acceptbttn = 'any'             'normal', 'extend', 'alternate', 'open', or 'any' 
%                                type of click the should be responded to 
%                                See 
% postpopcmd = 'axis(''auto'')'  command to execute in the new figure 
% sourceobj = gcbo               handle of the object to copy to the new figure 
% 
% U. Egert 6/98 
% 
% See also ADDPOPUCTRL
