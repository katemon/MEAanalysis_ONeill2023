Function synopsis

streamtypecount - returns indices to datastreams of a given type

Function comments

 
% [n,didx] = streamtypecount(D, streamtype) will search the datastream object <D> 
% for streams of a particular and return the number <n> and indices <didx> of 
% datastream starting with the string <streamtype>. If no matching streamname 
% was found <n> will be zero and <didx> will be empty. 
% 
% Example: 
% For a file where the first stream is a continuous record, streams 2 and 3 
% have the streamnames 'Spikes 1' and 'Spikes 2' respectively you may search 
% for all spike streams with 
% [n,didx] = streamtypecount(D, 'spikes'); 
% 
% The resulting n will be 2, didx will be [2 3]; 
% 
% U. Egert
