Function synopsis

CP2STRUCT - moves variables into a structure

Function comments

 
% CP2STRUCT(structname, inputvariables) moves the <inputvariables> 
%    into fields of <structname>. Input arguments 
%    must not be given as strings. If the structure <structname> 
%    does not exist it is created new. 
% 
% example 
%    if: 
%    a = 1:5; b = 'asdf'; 
% 
%    then: 
%    c = cp2struct(c, a, b) 
%    will return the structure c with the field c.a and c.b 
% 
% 
% U.Egert 8/97 
% 
% See also CP2STRUCT SETFIELD RMFIELD GETFIELD STRUCT
