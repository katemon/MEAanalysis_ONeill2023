Function synopsis

avgsweep - calculate means and standarddev. across dataset for LFP analysis

Function comments

 
% [meanSweeps,stdSweeps] = avgsweep(datasource, sweepsets) calculates the means across 
%    sets of data 'sheets' indexes in <sweepsets>. The data are 
%    retrieved from <datasource>, which can be the name of a 
%    (1) - datastream object, 
%    (2) - file suitable as datastream object 
%    (3) - a 3D array of size 64xNxP or 68xNxP 
% 
% With (1) and (2) the values in <sweepsets> must be indices to sweeps 
% in the datastream object. Values will be zero for electrodes not included 
% in the record. <sweepsets> can be cell arrays with sets of indices, in which 
% case averages/stds will be calculated separately for each cell. 
% 
% With (3) indices must point to the panes of 
% the 3D array, i.e. into the 3rd dimension. 
% 
% <meanSweeps> will be a matrix of size MxNxlength(sweepsets), where M is 
%    64 or 68 depending on the input data. 
% <stdSweeps> likewise holds the standarddeviations. These are calculate as 
%    if you were using the flag-option of std.m set to 1, i.e. using n instead 
%    of n-1. 
% 
% U. Egert 9/98
