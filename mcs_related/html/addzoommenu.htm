Function synopsis

addzoommenu - adds zoom menu that can be used with ginput

Function comments

 
% zmh = addzoommenu(targeth) checks for an exisiting Options 
% menu and appends/creates menu functions allowing autoscale, 
% zoom(2), zoom out and rubberband zoom on that figure. If 
% ginput is active (crosshair pointer) it is temporarily suspended 
% until the zoom OFF option is activated. On at least some 
% Linux systems interaction with ginput may not work. See 
% helpwin zoom for details on zoom functions. 
% 
% <targeth>          is the figure to which the menu will be added (gcf) 
% <zmh>              are handles to the menu entries 
% 
% (c) U. Egert 3/99 
% 
% See also ZOOM GINPUT AXIS
