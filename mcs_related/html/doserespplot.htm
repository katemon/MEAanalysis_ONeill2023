Function synopsis

doserespplot - creates a plot of dose/response curves

Function comments

 
% 
% [axh, lh] = doserespplot(ep_eval, epoch_data, varargin) creates dose response plots 
%              with a line for each datasset and a marker for each epoch. 
% 
% <ep_eval>         data matrix with one column for each dataset and one row for each epoch. 
%                    Each column will be a line in the plot. 
% <epoch_data>       a cell array defining the epochs in the format as returned by getepoch.m 
% 
% epoch_data = {<string descriptor>}  {<value>}   {<left ep. limit>}  {<right ep. limit>} 
% 
% example: 
%    'Bic'    [10]    [2.0576]    [3.3226] 
%    'Bic'    [20]    [5.1475]    [6.4124] 
%    'Bic'    [30]    [8.2788]    [9.3571] 
% 
% <varargin> modifies default setting with parameter/value pairs: 
%    'sortbyval'    [1] to sort the epochs by increasing 'doses', -1 for decreasing, 
%                   any other to omit sorting 
%    'legendstr'    []   a cell array of strings identifying the epochs 
%    'xtext'        string to use as xlabel 
%    'ytext'        string to use as ylabel 
%    'plotfunc'     plot type, legal values: ['plot'], 'semilogx', 'semilogy', 'loglog', 'bar'. 
% 
% 
% U. Egert 10/98
