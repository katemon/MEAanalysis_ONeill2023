Function synopsis

LOADGUICFG - loads a GUI configuration from a file

Function comments

 
% 
% loadguicfg(cfgnametag, fh) retrieves a GUI configuration structure from 
% a file and sets the GUI fields with tags corresponding to the 
% first-level fieldnames of the structure to the entries of these 
% fields. It restores GUI settings that have been saved with 
% SAVEGUICFG to the figure(fh). 
% If <fh> is omitted the current callback figure is used. 
% Use 'loadguicfg' to load such a file into a GUI. 
% 
% If an edit-field with a tag <cfgname> is found the structures name 
% is entered as the string of that field. 
% 
% U. Egert 6/98 
% 
% See also CFG2FIGUD GETGUIUD SETGUIUD LOADGUICFG
