Function synopsis

ELIMBL - used to eliminate double triggering during LFP analysis (MEATools)

Function comments

 
% ELIMDBL(inp, dist)                                                  [MEA/ab] 
%   deletes every second of two rows from matrix <inp> whose difference in the 
%   first column is less than <dist>.. 
% 
%   it's main purpose (for the time being) is the elimination of the second 
%   (and all of the following) stimulation artefact(s) while the first one 
%   is used as a trigger. It used to serve as a function to avoid douple 
%   triggering by stimulus artifact oscillations in lfp_extract. 
% 
%
