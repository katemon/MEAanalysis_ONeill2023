% matmovie = matlabmovie(M) 
% <matmovie> is a matrix of the type that can be played with movie.m. <M> 
% must be a matrix of N(1)*N(2) rows and any number of columns. The default assumes 
% 64 rows for an 8x8 matrix. The movie uses image.m to create pseudocolor 
% images of the values in the matrix <M>. Each column in M will give one 
% frame. Each frame will have the dimensions N(1)*N(2). The first and last columns 
% can be indicated using parameter value pairs (see below). 
% 
% Note that 
% (1) matlab movies require a large amount of memory. Make sure you use only 
% the data needed and that your virtual memory is sufficient. Ensure that 
% the image that should be part of the movie is not covered by another 
% window at any time while the movie is being built. 
% (2) parts of any window that overlap with the matlab figure you want 
%     to appear in the movie will be incorporated in the movie! 
% 
% 
% Parameter/value pairs used: 
% Parameter       value 
% 'start'         first column in M to show 
% 'stop'          last one 
% 'skip'          1 to show each column (def.), 2 for every second column, etc. 
% 'interp_value'  degree of interpolation for the patches between the grid positions 
%                 (def. 4) 
% 'interp_method' interpolation method to use. See help for interp2 for further information. 
%                 def is '*cubic'; 
% 'zlim'          color and z-axis limits, per default the min and max values of M are used. 
% 'ydir'         y-axis direction, def: 'reverse'. 
% 
% See also MOVIE IMAGE GETFRAME MOVIEBUILDER
