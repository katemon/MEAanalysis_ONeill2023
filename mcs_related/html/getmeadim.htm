Function synopsis

getmeadim - return the dimensions of a particular MEA type

Function comments

 
% getmeadim(meatype) returns a two-element vector giving the number 
% of rows and columns of a meatype. If not defined otherwise 
% it is assumed that <meatype> is a string in the format 'MxN' 
% where M gives the number of rows and N the number of columns. 
% 
% If <meatype> is 'nogrid', i.e. not a rectangular MEA, the 
% dimensions are returned [M, 1], where M is the total number 
% of channels indicated in the datastream object D (global). 
% 
% Note that a full dataset then must have M * N rows to fill 
% the MEA-layout grid. 
% 
% ********** EXTEND THIS ROUTINE FOR NEW MEA-LAYOUTS ****************** 
% 
% (c) U. Egert 2001
