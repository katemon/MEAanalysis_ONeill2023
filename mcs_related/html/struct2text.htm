Function synopsis

STRUCT2TEXT - converts simple structures to strings

Function comments

 
% outstr = struct2text(instruct) converts structures whose 1st level fields are 
% either single line strings or or vectors of numbers to matrices or cell arrays 
% of strings (default) with one line per field. Use the parameter/value pair 
% 'outstrtype', 'matrix'> to return a string matrix. Fieldnames and values are 
% separated by a delimiter given by e.g. <'delim', ' - '>, default is delimiter 
% 1 blank, i.e. ' '. 
% 
% STRUCT2TEXT does not support any other type of field values. 
% 
% U. Egert 6/98
