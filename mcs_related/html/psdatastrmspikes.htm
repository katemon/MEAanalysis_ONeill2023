psdatastrmspikes - extract spikes from an MCRack-datastream object

Function comments

 
% [msg]=psdatastrmspikes(infile, outfilepath, varargin) to reads data-files 
% with spike cutouts written with the MCS program 'MCRack' 
% in vers. 1.4 and up. 
% 
% Opens the file <infile> , reads and evaluates header and spike data, calculates 
% spike shape parameters for sorting and stores the data in <outfile>. 
% 
%    - PARMAT now contains principle component Eigenvalues and is calculated within rdspx85 
%    - pc vectors can be found in PCVECTORS with fields named like the electrodes 
%    - junkvalues, voltage thresholds and minim. voltages calculated for all traces/electrodes 
% 
% 
% U. Egert 2/1998
