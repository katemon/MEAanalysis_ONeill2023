Function synopsis

SPIKERATEBOX - user interface for spike rate analysis with MEATOOLS

Function comments

 
% only for use with MEATOOLS. Not a standalone function 
% The spike rate GUI supports the selection of electrode/subplot pairings, 
% rate calculation windows and increments, as well as display options for 
% the calculation of moving-window spike rate estimates based on MCRack 
% records of spike activity recorded with MEAs. The spike rate GUI is 
% part of the MEATools toolbox and called from its main menu. 
%
