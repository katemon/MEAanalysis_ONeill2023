PARTLINEAR - calculate linear indices into a matrix

% ei=partlinear(v,i,j,k,n), returns linear indices of ijk specified by col vectors i,j,k 
% *************************************************************** 
% *                                                             * 
% * Usage:                                                      * 
% *       v,n specify the source matrix. v is the matrix and    * 
% *       n the number of elements in v's first dimension.      * 
% *                                                             * 
% *       i,j,k column vectors containing 3d indices.           * 
% *             lenghth(i)==length(j)==length(k)                * 
% *                                                             * 
% *       ei, column vector containing linear indices of all    * 
% *           triples. length(ei)==length(i)                    * 
% *                                                             * 
% * Uses:                                                       * 
% *      rows, cols                                             * 
% *                                                             * 
% * See also:                                                   * 
% *           partposition                                      * 
% *           part, partindex                                   * 
% *                                                             *
