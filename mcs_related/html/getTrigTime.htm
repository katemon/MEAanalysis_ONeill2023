getTrigTime - retrieve the trigger position within a triggered sweep from a datastream object

Function comments

 
% TT = getTrigTime(varargin) tests wether the TimeWindow type of the 
% global datastream object D is 'Start on Trigger'. If so, the value 
% -1 * TimeWindow.Time1 is returned in <TT>. <TT> is empty otherwise. 
% 
% Optional arguments can be given through parameter value pairs: 
% parameter 
% 'filename'   instead of the global variable D the MCRack file entered as 
%              value is read. Note that this recreates D. 
% 'timetype'   'absolute' returns the absolute time of the trigger event 
%              within a particular sweep. The index of this sweep must 
%              be given in the 'sweepID' parameter/value pair. 
%              'relative' gives the trigger events times with respect to 
%              the start time of the sweeps indicated. 
%              'start_only' (default) returns the value of the 'TimeWindow.time1' 
%              property of the datastream object. 
% 'sweepID'    indicate the sweep number in queries for absolute trigger 
%              times. 'sweepID' can be a scalar or vector, default: [1]. 
% 
% The default settings (calls without input arguments) will return the time 
% of all trigger events within a  sweep relative to the start time of the sweep 
% (in ms). In case the trigger data stream was not recorded in the file the 
% value of the 'TimeWindow.time1' property of the datastream object is returned. 
%
