Function comments

 
% addpopupctrl(cmd, varargin) 
% creates/controls an 'options' menu item that enables/disables 
% a pop-up functions for grapics object that are double-clicked with 
% the mouse. 
% 
% If <cmd> is 'start' the menu is created and the buttondownfcn of all 
% graphics objects in the figure is set to 'popgco'. This is the same 
% as calling addpopupctrl without arguments. 
% 
% If <cmd> is 'togglepopup' the menu is switched off and the buttondownfcn 
% of all graphics objects is reset. 
% 
% Optionally, the target figure and the graphics objects whose 
% buttondownfcn is changed can be modified with parameter value pairs. 
% 
% pvps: 
% with <cmd> = 'start': 'figurehandle' - graphics handle to the figure that 
%                                        should be changed. 
%                       'objecthandle' - graphics handle to the object that 
%                                        should get pop-up functionality. 
% 
% (c) U. Egert 2002 
% 
% See also POPGCO
