Function synopsis

MAPTOIMG_BTTN - callback to map MEA electrodes on a bitmap image (MEATools component)

Function comments

 
% MAPTOIMG_BTTN is a callback in mcssummary.m. It is not a standalone function. 
% MAPTOIMG_BTTN starts a routine to plot the position of electrodes in RACKSETUP onto 
% bitmap image. The electrode display can be resized an moved by mouse action. 
% 
% Scaling is restricted to overall size, individual axes cannot be scaled. Default 
% aspect ratio is optimized for 1260x960 images and 8x8 MEA grids. Fine tuning needs to 
% be done manually. 
% 
% U. Egert 2/99 
%
