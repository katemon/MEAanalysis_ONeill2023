CREATESWEEPTEMPLATE - returns a mean across a set of MC-Rack sweeps

Function comments

 
% templmat = createswptemplate(dfilename, usesweeps) retrieves data from the MC-Racks 
%   sweep-data file given by <dfilename> (full path) and returns the mean of the values in the 
%   sweeps given by <usesweeps>. Values are given in uV, as 68*N matrices, and with all zeros 
%   on electrodes that were not recorded, i.e. that are not listed in the ChannelID field 
%   of the file.
