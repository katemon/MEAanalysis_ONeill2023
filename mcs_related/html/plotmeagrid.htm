plotmeagrid - creates a figure with the MEA layout

Function comments

 
% [figh, axh, doth] = plotmeagrid(axh, meatype, cid, varargin) 
% If called without input argument plotmeagrid will 
% plot a MEA layout pattern into the current axis as defined 
% by the properties of the global datastream object <D>. 
% 
% The optional arguments will modify this plot. 
% - If <axh> is given the plot will be added to the 
%   axis with the handle <axh>. 
% - <meatype> will change the layout of the MEA plotted, 
% - <cid> will plot only electrodes with these channelIDs 
% 
% (c) U. Egert 2001
