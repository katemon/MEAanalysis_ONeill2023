Function synopsis

startburstanalysis_batch - batch analysis of burst structure across several spike data files

Function comments

 
%  [nbursts, burstpos] = startburstanalysis_batch(threshold, spx_per_burst, ename, ClusterID, varargin) 
% 
% <threshold>        maximal inter-spike-interval for 2 spikes to belong to a burst 
% <spx_per_burst>    minimum number of spikes required to qualify a burst 
% <ename>            electrode name (string) 
% <ClusterID>        CLUSTERNUMBER to analyze 
% 
% 
% <nburst>           no. of bursts (cell array) 
% <burstpos>         indices to the spikes within the bursts (cell array) 
% 
% modifications by parameter/value pairs: 
% t2sec = 1e-6;                 % sampling time units in fraction of seconds 
% t2ms = t2sec*1e3;             % conversion factor from sampling time units to msec 
% timeresolution = 5;           % temporal resolution of the histogram given in msec 
% ChannelID = getcid(ename);    % electrode index 
% range = [];                   % time range to analyze 
% ipf = 1;                      % switches interpolation in the pseudocolor on 
% 
% (c) U. Egert 10/98 
% 
% See also BURSTANALYSIS
