Function comments

 
% c = pvstruct2cell(s) converts structure <s> to cell array <c> 
% For a = 1:2:(length(c)-1) c(a) gives the parameters and c(a+1) 
% the values for these parameters. 
% 
%   Example 
%         cfg.file = 'test.gdf'; cfg.type = 'spikes'; 
%         c_cfg = pvstruct2cell(cfg);
