Function synopsis

GUIcfg2cell - converts GUI-configuration structure to a cell array suitable for evaluation

Function comments

 
% outcell = GUIcfg2cell(cfg, varargin) uses the fieldnames of <cfg> as parameters and 
% takes the 'string' and 'value' fields of these subfields to build 
% a cell array holding fieldnames as parameters and the appropriate values. 
% This cell array is suitable as parameter/value-pair argument. 
% 
% <cfg> must comply to the structure style described in setguiud.m. 
% The optional cell argument <switchvals> determines how the checkbox/radiobutton 
% values 0 and 1 should be translated. <switchvals{1}> will be returned for 0. 
% Default setting is {'no', 'yes'}. 
% 
% GUIcfg2cell is a MEA-Tools component 
% U. Egert 6/98 
% 
% See also GETGUIUD SETGUIUD CFG2BASE CFG2FIGUD SETSELF CELLFUNC_CALL  GUICFG2STRUC
