Function synopsis

GETCID - finds ChannelID for electrodename

Function comments

 
% 
% cid = GETCID(electrodename) 
% assumes that the string electrodename is present in 
% RACKSETUP.ChannelNames_chosen which must match RACKSETUP.ChannelID. 
% If the string is found more than once the lowest value for 
% the corresponding position in RACKSETUP.ChannelID is returned. 
% Electrodename must be either a scalar, a string or a cell array of strings. 
% 
% Note: if the number of Analog Channels in a file is different from the 
% default, i.e. >3, the Name-to-Channel assignment is not fixed. You 
% then need to indicate the datastream object (usually the variable D) 
% as a parameter/value pair from which the assignment will be determined. 
% 
% optional parameter/value pairs 
% 'meatype'         '8x8' (default), 'nogrid' or other MxN formats 
% 'datastreamobj'   usually D 
% 
% 
% RACKSETUP must be global in the base workspace. 
% 
% U. Egert 12/97
