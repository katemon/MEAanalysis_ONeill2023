% propvalue = getobjprop(fh, idprop, idval, oprop) searches 
% objects whos property <idprop> has the value <idval>. The 
% value <propvalue> of property <oprop> of these objects is retrieved. 
% Unless an optional parent object handle <fh> is given the 
% current figure is searched. 
% 
% If no matching object can be found <propvalue> will be NaN. 
%
