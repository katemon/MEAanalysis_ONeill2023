CONTOURMOVIE - creates transparent animated contour plot

Function comments

 
% 
% syntax: 
% contourmovie(data, frlist, varargin) 
% <data> must be a 64xN matrix which will be shown as an 8x8 axis. 
% contourmovie will loop through <data> creating <flist> or N 
% frames if <frlist> is omitted. <frlist> is a column index list. 
% 
% Defaults can be adjusted by parameter value pairs: 
% 'targetaxis' (gca)    contours will be drawn into this axis 
% 'ipf'  (3)            interpolation factor (see interp2 for details) 
% 'caxlim' ([-50 30])   boundaries for color scaling 
% 'colorscale' ('jet')  a valid color map 
% 'ncontours'  (20)     number of contour levels to use 
%

%
