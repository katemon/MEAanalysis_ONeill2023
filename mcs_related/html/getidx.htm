Function synopsis

GETIDX - finds indices into SOURCECHANNELNUMBER for an electrode

Function comments

 
% 
% idxspx = getidx(ename, subpop, varargin) 
% returns the spike indices for a combination of electrode and 
% subpopulation identifiers. <ename> may be a string corresponding 
% to an electrode name, or an integer corresponding to a ChannelID. 
% <subpop> may be a scalar or vector, in which case the indices of 
% the spikes belonging to any of the listed subpopulation IDs on 
% the given electrode are returned. The index list may be further 
% cropped by defining start and end of a time range (in the units 
% of SPIKETIME) with a parameter value pair (in which case <subpop> 
% may be set to NaN to retrieve all subpopulations of an electrode): 
% 
% 'starttime'     earliest SPIKETIME allowed in units of SPIKETIME 
% 'endtime'       latest SPIKETIME allowed in units of SPIKETIME 
% 
%    global variables used: 
%    SOURCECHANNELNUMBER 
%    CLUSTERNUMBER 
%    SPIKETIME 
% 
%    U. Egert 12/97
