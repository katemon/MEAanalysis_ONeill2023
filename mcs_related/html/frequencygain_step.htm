Function synopsis

frequencygain_step - analyze the step response of MEA electrodes

Function comments

 
% frequencygain_step(file, sweepid, varargin) is used to calculate 
% the frequency gain function by analyzing the step responses of 
% MEA electrodes. The routine requires a sufficiently long sweep 
% (e.g. 2 sec) with the response of the MEA to a voltage step applied 
% to the bath. The psd.m function used is part of the signal processing 
% toolbox. For questions on technical implementation please ask the 
% Multichannelsystems people. 
% 
% <file>            MCRack data file 
% <sweepid>         integer defining the sweep to use or a 
%                   2-element vector of time in ms to use from 
%                   a continuous data record. 
% 
% parameter/pair modifiers, defaults in {}: 
% 
% 'windowtype'      a legal filter window, {'hanning'} 
% 'psd_samples'     no. of frequencies to estimate {2^10} 
% 'datarange'       2-element vector indicating first and 
%                   last element of the range to evaluate. 
%                   {[1 end]} 
% 'samplingrate'    {determined from the datastream object} 
% 
%
