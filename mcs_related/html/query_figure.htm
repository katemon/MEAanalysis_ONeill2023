query_figure - GUI to manage the figures currently open.

Function comments

 
% [fh, fname] = QUERY_FIGURE(cmd); creates a listbox with the figure 
% handles and figure names of the figures currently open, excluding 
% the MEA-Tools title figure and the listbox itself. Clicking on 
% a figure name will make this the current figure. 
% 
% If output arguments are used in the call to QUERY_FIGURE, <fh> 
% will hold the handle of the figure selected and <fname> will 
% hold its figure name. You can select only one figure at a time. 
% You then cannot access the command window without pressing the 
% 'done' or 'cancel' button. 
% 
% (c) U. Egert
