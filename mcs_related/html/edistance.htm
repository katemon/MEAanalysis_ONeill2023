Function synopsis

EDISTANCE - calculate the distance between a pattern and a sample

Function comments

 
% d = edistance(sample, pattern) 
% Calculates the eculidian distance between the pattern and the sample at 
% any position of the sample. <pattern> can be a skalar or a row vector. 
% <sample> can be a vector or a column matrix. In which the distance will 
% be calculated for each column in the matrix. <d> will be a vector or 2D-matrix 
% giving the euclidian distance, which is zero where the pattern matches the 
% sample exactly, non-zero in all other positions. 
% 
% U. Egert/S. Rotter 9/98
