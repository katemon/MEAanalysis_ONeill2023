removemenu - delete an item from a figure menu

Function comments

 
% removemenu(fh, menulabel, varargin) 
% where <fh> is the handle to the figure and <menulabel> the label to the 
% top level item (i.e. the figure menu). Additional arguements may be 
% added as strings indicating the submenu labels. Only the last submenu 
% corresponding to the last label in the list will be deleted, even when 
% the handles to any of the items are not visible. 
% Note that you need to include an amphersand (&) before the letter that 
% is underlined in each label. 
% 
% (c) U. Egert 2002
