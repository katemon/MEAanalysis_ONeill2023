% [AuV, sxvalues] = getanalog(file, refsweep); 
% file      - must be an MCRack datafile with triggered sweeps. 
% refsweep  - the sweep indices. If <refsweep> is a vector <AuV> will be 
%              a 3D matrix with one plane for each sweep. 
% AuV       - the retrieved voltage data 
% sxvalues  - the time axis for the sweep
