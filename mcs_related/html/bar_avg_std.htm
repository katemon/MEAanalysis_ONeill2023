Function synopsis

BAR_AVG_STD - bar histogram with error bars

Function comments

 
% [bh, lh] = bar_avg_std(avg, stdev, varargin) creates a plot of vertical 
% bars with error ranges indicated as symetric or positive vertical lines. 
% <avg> is MATRIX giving the bar length, <std> gives the 
% error line length. If <avg> and <std> are matrices a set of bars will 
% be drawn for each column. Patch handles are returned in <bh>, error bar 
% handles in <lh>. Plot details can be modified by parameter/value 
% pairs as follows (defaults): 
% 
% 'xdata' (1:length(avg))     x-axis positions of the bar centers, 
% 'marker' ('s')              marker type, see 'PLOT' for details, may be set to 'none' 
% 'markersize' (3)            size for said marker 
% 'width' (0.8)               relative width for the bars, bars contact each other with 1 
% 'symetric' (1)              1 for avg+/- standarddev., 0 for avg+standarddev. only. 
% 'lcolor' ('k')              error line color 
% 'fcolor', ('brgcmkyw')      the sequence of colors to use for the columns of <avg> 
% 
% (c) U. Egert 1/99 
% 
% See also ERRORBAR BAR BARH
