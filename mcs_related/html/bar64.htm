Function synopsis

BAR64 - create an 8x8 bar graph for MEA data

Function comments

 
% ahr = bar64(x, varargin) creates bar graphs in subplots on an 8x8 grid for each 
%			line in <x>. The plot can be modified with parameter values pairs for 
% 			<varargin>. 
% 
% Parameter              value 
% 'ymax'                 lower ylimit 
% 'ymax'                 upper ylimit 
% 'magnifyOnClick'       set buttondownfcn of each axis to magnify the subplot in a separate figure 
% 'xvalues'              x-axis values for the columns of 'x' 
% 'cid'                  subplot identifiers for the rows of <x> 
% 'fighandle'			handle to the target figure 
% 'yscdigits'			precision of the y-scale bar (default 1) 
% 'xscdigits'			precision of the x-scale bar (default 1) 
% 
% <ahr> are the axis handles. 
% 
% U. Egert 11/98
