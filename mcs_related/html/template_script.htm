Function synopsis

TEMPLATE_SCRIPT - create a sweep template from MCRack sweeps (e.g. as artifact model)

Function comments

 
%################################################################################################## 
% 
%              SCRIPT TO CREATE A SWEEP TEMPLATE FOR LFP-ANALYSIS OF MEA RECORDS 
% 
%    Contents: 
%                       - CREATE TEMPLATES 
%                       - UTILITIES 
% 
% 
%################################################################################################## 
% This script is used to extract a set of sweep data recorded with MCRack and MEAs, average the 
% data and thus create a template that may be used to correct for stimulus artifacts, subtract a 
% baseline response or similar. The script relies on the file format used by MCRack. The resulting 
% matrices are 64xN where N is the number of samples in a trace of the averaged sweeps. A second 
% set of commands is useful to view and verify the results and compare them to the original 
% data. 
% Don't forget to adjust the name of your template in the <template variable> field of the 
% LFP-Analysis I GUI! 
% 
% U. Egert 8/98
