Function synopsis

pc_matrix2 - calculates principle components for waveforms in a matrix

Function comments

 
% pc_matrix2(TRACES, use_npcs) 
%  calculates principle components for waveforms in a matrix with 
%  one spike per columns of the matrix <TRACES>. <use_npcs> limits 
%  the number of <pcvectors> returned. <contrib> is the contribution 
%  of each vector to the variance of the dataset. 
% 
%  Requires the global variable PARMAT as used in MEATools and stores 
% the result in the global variable PCVECTORS 
%

%
