Function synopsis

CHANMAP_ON_IMG - plots RACKSETUP.ChannelID electrodes on a bitmap image

Function comments

 
% chanmap_on_img(imgfile, varargin) retrieves <imgfile>, creates an image plot from 
% the data, and adds an axis showing the electrode positions defined by 
% RACKSETUP.ChannelID on top of the image. Valid image formats are given in imread.m. 
% The electrode axis can be moved and scale by mouse action. Scaling is restricted to 
% overall size, individual axes cannot be scaled. Default aspect ratio is optimized for 
% 1260x960 images and 8x8 MEA grids. Fine tuning needs to be done manually. 
% 
% U. Egert 2/99 
% 
% See also IMREAD CHAN_MAP IMGSURF
