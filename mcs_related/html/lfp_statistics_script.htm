Function synopsis

LFP_STATISTICS_SCRIPT - calculation of mean and average across sweeps in MCRack records

Function comments

 
%################################################################################################## 
% 
%              SCRIPT TO CALCULATE SOME STATISTICAL MEASURES OF MEA SWEEP-DATA 
% 
% Contents 
%              - DISPLAY FUNCTIONS 
%              - CALCULATE MEAN AND STANDARD DEVIATION ACROSS SWEEPS 
% 
% 
%################################################################################################## 
% 
% This script provides the commands necessary to calculate the mean across a set of sweeps 
% recorded with MCRack from MEAs. The result will be a 3D array named <meanSweeps> with 
% one row per electrode, one column per sampling point and one plane per set of averaged 
% sweeps. Note that the <sweepset>s listing the indices of the sweeps you want to average 
% across do not need to be continuous but can combine non-successive sweeps, e.g. 
% [1 5 3 81 6:10], which must then be enclosed in brackets. The result will be saved in 
% a variable named <meanSweeps>. A plot command at the bottom of this file helps 
% to view the result. 
% 
% This matrix type can be accessed by the plot64, movie and other utilities. 
% 
% U. Egert 8/98
