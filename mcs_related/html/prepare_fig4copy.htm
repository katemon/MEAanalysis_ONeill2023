% prepare_fig4copy(fh,action) changes some properties to 'prettify' the figure 
% The default value of <action> is 'settings'. If no argument is given, 
% a file open dialog box is given. If <fh> is missing, the action will be 
% performed on the current figure. 
% 
% <action> can be one of the following strings: 
% settings           -   parameters like font size, line width etc. will be 
%                        adapted according to default values, preference 
%                        structure 'prepare_fig4copy' or values in an 
%                        optional structure or alternatively as parameter/value pairs 
%                        The initial settings are stored in the 'userdata' 
%                        field of the figure 
% restore settings   -   restores the settings according to the stored 
%                        values 
% makeScalebar       -   turns the visibility of the axes off (except 
%                        colorbar) and draws a scalebar in the lower left 
%                        corner with appropriate labels. 
% makeScalebarDlg    -   same as makeScalebar but with a dialogbox for the 
%                        labels. 
% removeScalebar     -   removes the scalebar and turns the visibility of 
%                        the axes on again 
% 
% xlabel             -   [ms] % unit for x-axis of the scalebar 
% ylabel             -   [\muV] % unit for y-axis of the scalebar 
% PaperUnits         -   'centimeters'; 
% PaperOrientation   -   'portrait'; 
% PaperSize          -   [13.66 10.26]; %[width height] 
% PaperColor         -    [1 1 1]; 
% FontAngle          -    'normal'; 
% FontName           -    'Arial'; 
% FontSize           -    [16]; 
% FontUnits          -    'points'; 
% FontWeight         -    'normal'; 
% LineWidth          -    [2]; 
% TightBounding      -    1; 
% 
% See also PREPARE_SURF4COPY
