Function synopsis

RATECORR_PC - generates a pseudocolor rate correlation plot

Function comments

 
% ratecorr_pc is used to plot the changes of a xcorrelation function 
% generated across a moving window against experimental time. 
% NOTE: This function requires the signal processing toolbox! 
% 
% SYNTAX: 
% ratecorr_pc(xc, lags, rates, xgrid, A,B,ystep,ylag) 
% <xc>      the correlation matrix as generated by xcorr of the signal processing toolbox 
% <lags>    time lags returned by xcorr 
% <rates>   column matrix holding the spike rates 
% <xgrid>   vector with x-axis values 
% <A>       column ID of the first set of rates and correlations 
% <B>       column ID of the second set of rates 
% <ystep>   time interval between two <xc> rows and <lags> values 
% <ylag>    the extend of the lag range to show on the y-axis, 
%           ylim will be [-ylag ylag] 
% 
% (c) U. Egert 1/1999 
% 
% See also RATECORRSCRIPT XCORR
