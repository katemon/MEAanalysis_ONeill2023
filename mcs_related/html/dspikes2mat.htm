Function synopsis

dspikes2mat - retrieve spike cutouts to a column matrix (MEATools)

Function comments

 
% 
% TRACES  = dspikes2mat(idx, tracetype, varargin) retrieves spike cutouts recorded 
% with MCRack v. 1.3 & up, to a matrix in the base workspace. <idx> is a vector of 
% indices into the global variable <spkpointer>, <tracetype> can be 
% 'raw' or 'filtered' resulting in either unfiltered or Savitzky-Golay 
% filtered cutouts. 
% 
% If called without input arguments an input dialog box is opend requesting 
% the necessary information. 
% 
% NOTE: 
% It is currently required that you have parsed the file beforehand once using 
% the 'parse spike files' menu of the MEA-Tools GUI. 
% 
% (c) U. Egert 3/99 
% 
% See also TRACE2MAT
