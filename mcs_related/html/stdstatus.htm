% h = stdstatus(infotext, fh) creates a status text field 
% in the lower left corner of the figure identified by 
% the handle <fh>. The string of the textfield is set 
% to <infotext>. stdstatus returns the handle <h> to 
% the object. The string of <h> can be accessed by 
% setstatus and getstatus.

% See also GETSTATUS SETSTATUS
