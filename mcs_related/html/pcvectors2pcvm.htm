Function synopsis

pcvectors2pcvm - create a principle component matrix from PCVECTORS (MEATools component)

Function comments

 
% pcvm = pcvectors2pcvm uses the structure variable <vecvar> (usually PCVECTORS) to generate a 3D pc vector 
% matrix by reformatting <vecvar>. <vecvar> must be a structure with each field corresponding to matrix 
% of principle component vectors for one electrode. This is resolved to have one principle component vector 
% in each column, with one plane per elctrode. All vectors must have equal length. 
%
