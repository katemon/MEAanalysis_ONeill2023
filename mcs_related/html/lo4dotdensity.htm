Function synopsis

lo4dotdensity - run a laplace operator across a matrix

Function comments

 
% C = lo4dotdensity(c, loparams) will calculate a 
% laplace operator as the difference of two 2D Gauss functions G1 and 
% G2 with a size loparams(3) squared and standard deviations 
% <loparams(1)> and <loparams(2)>. The result is G2-G1. 
% 
% The input matrix <c> will be expanded by a rim of zeros with the 
% width of the operator. Note that this restricts the matrix such 
% that its borders should be zero to avoid boundary effects. 
% The size of the operator will be adjusted to odd numbers with the 
% peak aligned to the center. 
%
